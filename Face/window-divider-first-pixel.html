Face: window-divider-first-pixel (sample)

Documentation:
Basic face for first pixel line/column of window dividers. When a divider is at least 3 pixels wide, its first pixel line/column is drawn with the foreground of this face. If you do not want to accentuate the first pixel line/column, set this to the same as `window-divider' face.

Defined in `faces.el'.
