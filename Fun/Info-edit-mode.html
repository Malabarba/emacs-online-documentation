Info-edit-mode is an interactive compiled Lisp function in `info.el'.


(Info-edit-mode)

Parent mode: `text-mode'.

This function is obsolete since 24.4;
editing Info nodes by hand is not recommended.

Major mode for editing the contents of an Info node.
Like text mode with the addition of `Info-cease-edit'
which returns to Info mode for browsing.
key binding
--- -------

C-c Prefix Command
ESC Prefix Command

C-c C-c Info-cease-edit

C-M-i ispell-complete-word



In addition to any hooks its parent mode `text-mode' might have run,
this mode runs the hook `Info-edit-mode-hook', as the final step
during initialization.
