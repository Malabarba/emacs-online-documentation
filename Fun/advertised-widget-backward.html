advertised-widget-backward is an alias for `widget-backward' in `wid-edit.el'.


(advertised-widget-backward ARG)

This function is obsolete since 23.2;
use `widget-backward' instead.

Move point to the previous field or button.
With optional ARG, move across that many fields.
