advertised-xscheme-send-previous-expression is an alias for `xscheme-send-previous-expression' in `xscheme.el'.


(advertised-xscheme-send-previous-expression)

This function is obsolete since 23.2;
use `xscheme-send-previous-expression' instead.

Send the expression to the left of `point' to the Scheme process.
