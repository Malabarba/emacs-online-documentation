ansi-color-unfontify-region is an alias for `font-lock-default-unfontify-region' in `ansi-color.el'.

(ansi-color-unfontify-region BEG END)

This function is obsolete since 24.1;
use `font-lock-default-unfontify-region' instead.

Unfontify the text between BEG and END.
This function is the default `font-lock-unfontify-region-function'.
