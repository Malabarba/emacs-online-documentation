apropos-mode is an interactive compiled Lisp function in `apropos.el'.

(apropos-mode)

Parent mode: `special-mode'.

Major mode for following hyperlinks in output of apropos commands.

key binding
--- -------

TAB forward-button
RET apropos-follow
ESC Prefix Command
SPC scroll-up-command
- negative-argument
0 .. 9 digit-argument
< beginning-of-buffer
> end-of-buffer
? describe-mode
g revert-buffer
h describe-mode
q quit-window
DEL scroll-down-command
backward-button
Prefix Command

C-M-i backward-button



In addition to any hooks its parent mode `special-mode' might have run,
this mode runs the hook `apropos-mode-hook', as the final step
during initialization.
