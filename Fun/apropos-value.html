apropos-value is an interactive autoloaded Lisp function in `apropos.el'.

It is bound to <menu-bar> <help-menu> <search-documentation> <find-option-by-value>.

(apropos-value PATTERN &optional DO-ALL)

Show all symbols whose value's printed representation matches PATTERN.
PATTERN can be a word, a list of words (separated by spaces),
or a regexp (using some regexp special characters). If it is a word,
search for matches for that word as a substring. If it is a list of words,
search for matches for any two (or more) of those words.

With C-u prefix, or if `apropos-do-all' is non-nil, also looks
at function definitions (arguments, documentation and body) and at the
names and values of properties.

Returns list of symbols and values found.
