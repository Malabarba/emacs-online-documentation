artist-find-direction is a compiled Lisp function in `artist.el'.

(artist-find-direction X1 Y1 X2 Y2)

Find the direction from point X1,Y1 to X2,Y2.
Returns a DIRECTION, a number 0--7, coded as follows:

5 6 7
\ | /
4 - * - 0
/ | \
3 2 1
