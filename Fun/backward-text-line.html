backward-text-line is an alias for `nroff-backward-text-line' in `nroff-mode.el'.

(backward-text-line &optional CNT)

This function is obsolete since 22.1;
use `nroff-backward-text-line' instead.

Go backward one nroff text line, skipping lines of nroff requests.
An argument is a repeat count; negative means move forward.
