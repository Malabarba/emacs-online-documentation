--- title: "bug-reference-mode" layout: doc --- bug-reference-mode is an interactive autoloaded compiled Lisp function in `bug-reference.el'.


(bug-reference-mode &optional ARG)

Toggle hyperlinking bug references in the buffer (Bug Reference mode).
With a prefix argument ARG, enable Bug Reference mode if ARG is
positive, and disable it otherwise. If called from Lisp, enable
the mode if ARG is omitted or nil.
