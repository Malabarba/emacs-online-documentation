checkdoc-ispell-current-buffer is an interactive autoloaded compiled Lisp function in `checkdoc.el'.

(checkdoc-ispell-current-buffer &optional TAKE-NOTES)

Check the style and spelling of the current buffer.
Calls `checkdoc-current-buffer' with spell-checking turned on.
Prefix argument TAKE-NOTES is the same as for `checkdoc-current-buffer'
