checkdoc-ispell-message-interactive is an interactive autoloaded compiled Lisp function in `checkdoc.el'.

(checkdoc-ispell-message-interactive &optional TAKE-NOTES)

Check the style and spelling of message text interactively.
Calls `checkdoc-message-interactive' with spell-checking turned on.
Prefix argument TAKE-NOTES is the same as for `checkdoc-message-interactive'
