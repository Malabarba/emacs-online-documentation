--- title: "checkdoc-output-mode" layout: doc --- checkdoc-output-mode is an interactive compiled Lisp function in `checkdoc.el'.


(checkdoc-output-mode)

Parent mode: `compilation-mode'.

Set up the major mode for the buffer containing the list of errors.

In addition to any hooks its parent mode `compilation-mode' might have run,
this mode runs the hook `checkdoc-output-mode-hook', as the final step
during initialization.
