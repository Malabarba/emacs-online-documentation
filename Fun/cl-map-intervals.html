cl-map-intervals is an alias for `cl--map-intervals' in `cl.el'.

(cl-map-intervals CL-FUNC &optional CL-WHAT CL-PROP CL-START CL-END)

This function is obsolete since 24.3;
use `cl--map-intervals' instead.

Not documented.
