condition-case-no-debug is an alias for `condition-case-unless-debug' in `subr.el'.

(condition-case-no-debug VAR BODYFORM &rest HANDLERS)

This function is obsolete since 24.1;
use `condition-case-unless-debug' instead.

Like `condition-case' except that it does not catch anything when debugging.
More specifically if `debug-on-error' is set, then it does not catch any signal.
