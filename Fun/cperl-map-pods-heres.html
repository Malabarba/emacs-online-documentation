--- title: "cperl-map-pods-heres" layout: doc --- cperl-map-pods-heres is a compiled Lisp function in `cperl-mode.el'.

(cperl-map-pods-heres FUNC &optional PROP S END)

Executes a function over regions of pods or here-documents.
PROP is the text-property to search for; default to `in-pod'. Stop when
function returns nil.
