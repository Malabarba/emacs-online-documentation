crm-minibuffer-completion-help is an alias for `crm-completion-help' in `crm.el'.


(crm-minibuffer-completion-help)

This function is obsolete since 23.1;
use `crm-completion-help' instead.

Display a list of possible completions of the current minibuffer element.
