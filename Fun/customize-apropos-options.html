customize-apropos-options is an interactive autoloaded compiled Lisp function in `cus-edit.el'.

It is bound to <menu-bar> <options> <customize> <customize-apropos-options>.

(customize-apropos-options REGEXP &optional IGNORED)

Customize all loaded customizable options matching REGEXP.
