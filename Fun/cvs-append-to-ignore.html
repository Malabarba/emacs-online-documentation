cvs-append-to-ignore is an alias for `vc-cvs-append-to-ignore' in `pcvs.el'.

(cvs-append-to-ignore DIR STR &optional OLD-DIR)

This function is obsolete since 24.4;
use `vc-cvs-append-to-ignore' instead.

In DIR, add STR to the .cvsignore file.
If OLD-DIR is non-nil, then this is a directory that we don't want
to hear about anymore.
