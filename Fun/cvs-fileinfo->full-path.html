cvs-fileinfo->full-path is an alias for `cvs-fileinfo->full-name' in `pcvs-info.el'.

(cvs-fileinfo->full-path FILEINFO)

This function is obsolete since 22.1;
use `cvs-fileinfo->full-name' instead.

Return the full path for the file that is described in FILEINFO.
