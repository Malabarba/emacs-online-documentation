--- title: "cvs-mode-tag" layout: doc --- cvs-mode-tag is an interactive compiled Lisp function in `pcvs.el'.


(cvs-mode-tag TAG &optional FLAGS)

Run `cvs tag TAG' on all selected files.
With prefix argument, prompt for cvs flags.
By default this can only be used on directories.
Use M-x cvs-mode-force-command or change `cvs-force-dir-tag' if you need
to use it on individual files.
