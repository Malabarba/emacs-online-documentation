cvs-mode-toggle-marks is an interactive compiled Lisp function in `pcvs.el'.


(cvs-mode-toggle-marks ARG)

Toggle whether the next CVS command uses marks.
See `cvs-prefix-set' for further description of the behavior.
C-u 1 selects `force-marks',
C-u 2 selects `ignore-marks',
C-u 3 selects `toggle-marks'.
