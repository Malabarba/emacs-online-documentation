cvs-string-prefix-p is an alias for `string-prefix-p' in `pcvs-util.el'.

(cvs-string-prefix-p STR1 STR2 &optional IGNORE-CASE)

This function is obsolete since 24.3;
use `string-prefix-p' instead.

Return non-nil if STR1 is a prefix of STR2.
If IGNORE-CASE is non-nil, the comparison is done without paying attention
to case differences.
