--- title: "dbus-introspect-get-argument" layout: doc --- dbus-introspect-get-argument is a compiled Lisp function in `dbus.el'.

(dbus-introspect-get-argument BUS SERVICE PATH INTERFACE NAME ARG)

Return argument ARG as XML object.
NAME must be a "method" or "signal" object. ARG must be a string,
element of the list returned by `dbus-introspect-get-argument-names'.
