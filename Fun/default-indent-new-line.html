--- title: "default-indent-new-line" layout: doc --- default-indent-new-line is an interactive compiled Lisp function in `simple.el'.


(default-indent-new-line &optional SOFT)

Break line at point and indent.
If a comment syntax is defined, call `comment-indent-new-line'.

The inserted newline is marked hard if variable `use-hard-newlines' is true,
unless optional argument SOFT is non-nil.
