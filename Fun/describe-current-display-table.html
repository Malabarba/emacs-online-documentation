describe-current-display-table is an interactive autoloaded compiled Lisp function in `disp-table.el'.

It is bound to <menu-bar> <help-menu> <describe> <describe-current-display-table>.

(describe-current-display-table)

Describe the display table in use in the selected window and buffer.
