describe-distribution is an interactive compiled Lisp function in `help.el'.

It is bound to C-h C-o, <f1> C-o, <help> C-o, <menu-bar> <help-menu> <getting-new-versions>.

(describe-distribution)

Display info on how to obtain the latest version of GNU Emacs.
