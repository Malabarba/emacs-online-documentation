describe-no-warranty is an interactive compiled Lisp function in `help.el'.

It is bound to C-h C-w, <f1> C-w, <help> C-w, <menu-bar> <help-menu> <describe-no-warranty>.

(describe-no-warranty)

Display info on all the kinds of warranty Emacs does NOT have.
