--- title: "dframe-temp-buffer-show-function" layout: doc --- dframe-temp-buffer-show-function is a compiled Lisp function in `dframe.el'.

(dframe-temp-buffer-show-function BUFFER)

Placed in the variable `temp-buffer-show-function' in dedicated frames.
If a user requests help using C-h <Key> the temp BUFFER will be
redirected into a window on the attached frame.
