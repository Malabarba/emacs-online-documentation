--- title: "diary-islamic-mark-entries" layout: doc --- diary-islamic-mark-entries is an autoloaded Lisp function in `cal-islam.el'.

(diary-islamic-mark-entries)

Mark days in the calendar window that have Islamic date diary entries.
Marks each entry in `diary-file' (or included files) visible in the calendar
window. See `diary-islamic-list-entries' for more information.
