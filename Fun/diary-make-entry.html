diary-make-entry is an autoloaded compiled Lisp function in `diary-lib.el'.

(diary-make-entry STRING &optional NONMARKING FILE)

Insert a diary entry STRING which may be NONMARKING in FILE.
If omitted, NONMARKING defaults to nil and FILE defaults to
`diary-file'.
