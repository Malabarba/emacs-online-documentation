diary-rosh-hodesh is an alias for `diary-hebrew-rosh-hodesh' in `cal-hebrew.el'.

(diary-rosh-hodesh &optional MARK)

This function is obsolete since 23.1;
use `diary-hebrew-rosh-hodesh' instead.

Rosh Hodesh diary entry.
Entry applies if date is Rosh Hodesh, the day before, or the Saturday before.

An optional parameter MARK specifies a face or single-character string to
use when highlighting the day in the calendar.
