diary-sabbath-candles is an alias for `diary-hebrew-sabbath-candles' in `cal-hebrew.el'.

(diary-sabbath-candles &optional MARK)

This function is obsolete since 23.1;
use `diary-hebrew-sabbath-candles' instead.

Local time of candle lighting diary entry--applies if date is a Friday.
No diary entry if there is no sunset on that date. Uses
`diary-hebrew-sabbath-candles-minutes'.

An optional parameter MARK specifies a face or single-character string to
use when highlighting the day in the calendar.
