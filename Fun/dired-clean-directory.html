--- title: "dired-clean-directory" layout: doc --- dired-clean-directory is an interactive autoloaded compiled Lisp function in `dired-aux.el'.


(dired-clean-directory KEEP)

Flag numerical backups for deletion.
Spares `dired-kept-versions' latest versions, and `kept-old-versions' oldest.
Positive prefix arg KEEP overrides `dired-kept-versions';
Negative prefix arg KEEP overrides `kept-old-versions' with KEEP made positive.

To clear the flags on these files, you can use M-x dired-flag-backup-files
with a prefix argument.
