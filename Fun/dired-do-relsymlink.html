--- title: "dired-do-relsymlink" layout: doc --- dired-do-relsymlink is an interactive autoloaded compiled Lisp function in `dired-x.el'.


(dired-do-relsymlink &optional ARG)

Relative symlink all marked (or next ARG) files into a directory.
Otherwise make a relative symbolic link to the current file.
This creates relative symbolic links like

foo -> ../bar/foo

not absolute ones like

foo -> /ugly/file/name/that/may/change/any/day/bar/foo

For absolute symlinks, use M-x dired-do-symlink.
