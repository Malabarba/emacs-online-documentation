dired-extra-startup is an interactive compiled Lisp function in `dired-x.el'.

(dired-extra-startup)

Automatically put on `dired-mode-hook' to get extra Dired features:

V -- run mail on folder (see `dired-bind-vm')
I -- run info on file
N -- run man on file
F -- visit all marked files simultaneously
M-o -- toggle omitting of files
M-( -- mark by Lisp expression

To see the options you can set, use M-x customize-group RET dired-x RET.
See also the functions:
`dired-flag-extension'
`dired-virtual'
`dired-jump'
`dired-man'
`dired-vm'
`dired-rmail'
`dired-info'
`dired-do-find-marked-files'
