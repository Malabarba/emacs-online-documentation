dirtrack-debug-toggle is an alias for `dirtrack-debug-mode' in `dirtrack.el'.

(dirtrack-debug-toggle &optional ARG)

This function is obsolete since 23.1;
use `dirtrack-debug-mode' instead.

Toggle Dirtrack debugging.
With a prefix argument ARG, enable Dirtrack debugging if ARG is
positive, and disable it otherwise. If called from Lisp, enable
the mode if ARG is omitted or nil.
