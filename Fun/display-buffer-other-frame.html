--- title: "display-buffer-other-frame" layout: doc --- display-buffer-other-frame is an interactive compiled Lisp function in `window.el'.

It is bound to C-x 5 C-o.

(display-buffer-other-frame BUFFER)

Display buffer BUFFER preferably in another frame.
This uses the function `display-buffer' as a subroutine; see
its documentation for additional customization information.
