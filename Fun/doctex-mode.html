--- title: "doctex-mode" layout: doc --- doctex-mode is an interactive autoloaded compiled Lisp function in `tex-mode.el'.


(doctex-mode)

Parent mode: `latex-mode'.

Major mode to edit DocTeX files.

In addition to any hooks its parent mode `latex-mode' might have run,
this mode runs the hook `doctex-mode-hook', as the final step
during initialization.
