dun-mode is an interactive compiled Lisp function in `dunnet.el'.


(dun-mode)

Parent mode: `text-mode'.

Major mode for running dunnet.

In addition to any hooks its parent mode `text-mode' might have run,
this mode runs the hook `dun-mode-hook', as the final step
during initialization.
