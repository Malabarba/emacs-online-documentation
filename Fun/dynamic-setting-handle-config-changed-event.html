--- title: "dynamic-setting-handle-config-changed-event" layout: doc --- dynamic-setting-handle-config-changed-event is an interactive compiled Lisp function in `dynamic-setting.el'.


(dynamic-setting-handle-config-changed-event EVENT)

Handle config-changed-event on the display in EVENT.
Changes can be
The monospace font. If `font-use-system-font' is nil, the font
is not changed.
The normal font.
Xft parameters, like DPI and hinting.
The Gtk+ theme name.
The tool bar style.
