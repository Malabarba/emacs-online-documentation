--- title: "ebnf-syntax-file" layout: doc --- ebnf-syntax-file is an interactive autoloaded compiled Lisp function in `ebnf2ps.el'.


(ebnf-syntax-file FILE &optional DO-NOT-KILL-BUFFER-WHEN-DONE)

Do a syntactic analysis of the named FILE.

If optional arg DO-NOT-KILL-BUFFER-WHEN-DONE is non-nil, the buffer isn't
killed after syntax checking.

See also `ebnf-syntax-buffer'.
