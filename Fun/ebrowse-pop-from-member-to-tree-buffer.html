ebrowse-pop-from-member-to-tree-buffer is an interactive compiled Lisp function in `ebrowse.el'.


(ebrowse-pop-from-member-to-tree-buffer ARG)

Pop from a member buffer to the matching tree buffer.
Switch to the buffer if prefix ARG. If no tree buffer exists,
make one.
