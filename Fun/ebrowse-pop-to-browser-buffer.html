ebrowse-pop-to-browser-buffer is an interactive compiled Lisp function in `ebrowse.el'.

It is bound to C-c RET b.

(ebrowse-pop-to-browser-buffer ARG)

Pop to a browser buffer from any other buffer.
Pop to member buffer if no prefix ARG, to tree buffer otherwise.
