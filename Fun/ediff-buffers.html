ediff-buffers is an interactive autoloaded compiled Lisp function in `ediff.el'.

It is bound to <menu-bar> <tools> <compare> <ediff-buffers>.

(ediff-buffers BUFFER-A BUFFER-B &optional STARTUP-HOOKS JOB-NAME)

Run Ediff on a pair of buffers, BUFFER-A and BUFFER-B.
