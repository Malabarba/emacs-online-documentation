ediff-buffers3 is an interactive autoloaded compiled Lisp function in `ediff.el'.

It is bound to <menu-bar> <tools> <compare> <ediff-buffers3>.

(ediff-buffers3 BUFFER-A BUFFER-B BUFFER-C &optional STARTUP-HOOKS JOB-NAME)

Run Ediff on three buffers, BUFFER-A, BUFFER-B, and BUFFER-C.
