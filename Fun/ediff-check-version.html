--- title: "ediff-check-version" layout: doc --- ediff-check-version is a compiled Lisp function in `ediff-init.el'.

(ediff-check-version OP MAJOR MINOR &optional TYPE-OF-EMACS)

This function is obsolete since 23.1;
use `version<' instead.

Check the current version against MAJOR and MINOR version numbers.
The comparison uses operator OP, which may be any of: =, >, >=, <, <=.
TYPE-OF-EMACS is either 'xemacs or 'emacs.
