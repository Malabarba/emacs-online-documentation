--- title: "ediff-directory-revisions" layout: doc --- ediff-directory-revisions is an interactive autoloaded compiled Lisp function in `ediff.el'.

It is bound to <menu-bar> <tools> <compare> <ediff-dir-revision>.

(ediff-directory-revisions DIR1 REGEXP)

Run Ediff on a directory, DIR1, comparing its files with their revisions.
The second argument, REGEXP, is a regular expression that filters the file
names. Only the files that are under revision control are taken into account.
