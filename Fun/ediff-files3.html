ediff-files3 is an interactive autoloaded compiled Lisp function in `ediff.el'.

It is bound to <menu-bar> <tools> <compare> <ediff-files3>.

(ediff-files3 FILE-A FILE-B FILE-C &optional STARTUP-HOOKS)

Run Ediff on three files, FILE-A, FILE-B, and FILE-C.
