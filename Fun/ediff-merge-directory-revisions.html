ediff-merge-directory-revisions is an interactive autoloaded compiled Lisp function in `ediff.el'.

It is bound to <menu-bar> <tools> <ediff-merge> <ediff-merge-dir-revisions>.

(ediff-merge-directory-revisions DIR1 REGEXP &optional MERGE-AUTOSTORE-DIR)

Run Ediff on a directory, DIR1, merging its files with their revisions.
The second argument, REGEXP, is a regular expression that filters the file
names. Only the files that are under revision control are taken into account.
