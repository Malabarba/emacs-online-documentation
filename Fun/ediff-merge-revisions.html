ediff-merge-revisions is an interactive autoloaded compiled Lisp function in `ediff.el'.

It is bound to <menu-bar> <tools> <ediff-merge> <ediff-merge-revisions>.

(ediff-merge-revisions &optional FILE STARTUP-HOOKS MERGE-BUFFER-FILE)

Run Ediff by merging two revisions of a file.
The file is the optional FILE argument or the file visited by the current
buffer.
