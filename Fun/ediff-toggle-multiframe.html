ediff-toggle-multiframe is an interactive autoloaded compiled Lisp function in `ediff-util.el'.

It is bound to <menu-bar> <tools> <compare> <ediff-misc> <emultiframe>.

(ediff-toggle-multiframe)

Switch from multiframe display to single-frame display and back.
To change the default, set the variable `ediff-window-setup-function',
which see.
