edt-cut-or-copy is an interactive compiled Lisp function in `edt.el'.


(edt-cut-or-copy)

Cuts (or copies) selected text to kill ring.
Cuts selected text if `buffer-read-only' is nil.
Copies selected text if `buffer-read-only' is t.
