--- title: "electric-nroff-newline" layout: doc --- electric-nroff-newline is an alias for `nroff-electric-newline' in `nroff-mode.el'.


(electric-nroff-newline ARG)

This function is obsolete since 22.1;
use `nroff-electric-newline' instead.

Insert newline for nroff mode; special if nroff-electric mode.
In `nroff-electric-mode', if ending a line containing an nroff opening request,
automatically inserts the matching closing request after point.
