electric-perl-terminator is an alias for `perl-electric-terminator' in `perl-mode.el'.


(electric-perl-terminator ARG)

This function is obsolete since 22.1;
use `perl-electric-terminator' instead.

Insert character and maybe adjust indentation.
If at end-of-line, and not in a comment or a quote, correct the indentation.
