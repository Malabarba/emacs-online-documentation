--- title: "elisp-byte-code-mode" layout: doc --- elisp-byte-code-mode is an interactive compiled Lisp function in `elisp-mode.el'.


(elisp-byte-code-mode)

Parent mode: `emacs-lisp-mode'.

Major mode for *.elc files.

In addition to any hooks its parent mode `emacs-lisp-mode' might have run,
this mode runs the hook `elisp-byte-code-mode-hook', as the final step
during initialization.
