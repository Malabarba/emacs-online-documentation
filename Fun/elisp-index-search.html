elisp-index-search is an interactive compiled Lisp function in `menu-bar.el'.

It is bound to <menu-bar> <help-menu> <search-documentation> <lookup-subject-in-elisp-manual>.

(elisp-index-search TOPIC)

Look up TOPIC in the indices of the Emacs Lisp Reference Manual.
