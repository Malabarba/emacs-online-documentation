--- title: "emerge-edit-mode" layout: doc --- emerge-edit-mode is an interactive compiled Lisp function in `emerge.el'.


(emerge-edit-mode)

Set edit mode, for Emerge.
In this mode ordinary Emacs commands are available, and Emerge commands
must be prefixed with
Uses keymap `emerge-fast-keymap', which is not currently defined.
M-x emerge-basic-keymap.
