emerge-fast-mode is an interactive compiled Lisp function in `emerge.el'.


(emerge-fast-mode)

Set fast mode, for Emerge.
In this mode ordinary Emacs commands are disabled, and Emerge commands
need not be prefixed with
Uses keymap `emerge-fast-keymap', which is not currently defined.
M-x emerge-basic-keymap.
