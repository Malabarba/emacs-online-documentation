--- title: "epa-decrypt-file" layout: doc --- epa-decrypt-file is an interactive autoloaded compiled Lisp function in `epa.el'.

It is bound to <menu-bar> <tools> <encryption-decryption> <decrypt-file>.

(epa-decrypt-file DECRYPT-FILE &optional PLAIN-FILE)

Decrypt DECRYPT-FILE into PLAIN-FILE.
If you do not specify PLAIN-FILE, this functions prompts for the value to use.
