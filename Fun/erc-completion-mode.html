--- title: "erc-completion-mode" layout: doc --- erc-completion-mode is an alias for `erc-pcomplete-mode' in `erc-pcomplete.el'.


(erc-completion-mode &optional ARG)

Toggle ERC pcomplete mode.
With a prefix argument ARG, enable pcomplete if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
In ERC Completion mode, the TAB key does completion whenever possible.
