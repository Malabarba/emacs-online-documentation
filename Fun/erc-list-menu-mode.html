erc-list-menu-mode is an interactive compiled Lisp function in `erc-list.el'.


(erc-list-menu-mode)

Parent mode: `special-mode'.

Major mode for editing a list of irc channels.

In addition to any hooks its parent mode `special-mode' might have run,
this mode runs the hook `erc-list-menu-mode-hook', as the final step
during initialization.
