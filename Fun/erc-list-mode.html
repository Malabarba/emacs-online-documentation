--- title: "erc-list-mode" layout: doc --- erc-list-mode is an interactive autoloaded compiled Lisp function in `erc-list.el'.


(erc-list-mode &optional ARG)

Toggle ERC list mode.
With a prefix argument ARG, enable list if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
List channels nicely in a separate buffer.
