erc-menu-mode is an interactive autoloaded compiled Lisp function in `erc-menu.el'.


(erc-menu-mode &optional ARG)

Toggle ERC menu mode.
With a prefix argument ARG, enable menu if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
Enable a menu in ERC buffers.
