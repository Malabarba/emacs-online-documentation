erc-move-to-prompt-mode is an interactive compiled Lisp function in `erc-goodies.el'.


(erc-move-to-prompt-mode &optional ARG)

Toggle ERC move-to-prompt mode.
With a prefix argument ARG, enable move-to-prompt if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
This mode causes the point to be moved to the prompt when typing text.
