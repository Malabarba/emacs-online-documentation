erc-noncommands-mode is an interactive compiled Lisp function in `erc-goodies.el'.


(erc-noncommands-mode &optional ARG)

Toggle ERC noncommands mode.
With a prefix argument ARG, enable noncommands if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
This mode distinguishes non-commands.
Commands listed in `erc-insert-this' know how to display
themselves.
