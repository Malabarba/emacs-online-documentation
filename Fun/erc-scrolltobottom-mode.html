erc-scrolltobottom-mode is an interactive compiled Lisp function in `erc-goodies.el'.

(erc-scrolltobottom-mode &optional ARG)

Toggle ERC scrolltobottom mode.
With a prefix argument ARG, enable scrolltobottom if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
This mode causes the prompt to stay at the end of the window.
