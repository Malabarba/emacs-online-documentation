--- title: "erc-timestamp-mode" layout: doc --- erc-timestamp-mode is an alias for `erc-stamp-mode' in `erc-stamp.el'.


(erc-timestamp-mode &optional ARG)

Toggle ERC stamp mode.
With a prefix argument ARG, enable stamp if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
This mode timestamps messages in the channel buffers.
