erc-unmorse-mode is an interactive compiled Lisp function in `erc-goodies.el'.


(erc-unmorse-mode &optional ARG)

Toggle ERC unmorse mode.
With a prefix argument ARG, enable unmorse if ARG is positive,
and disable it otherwise. If called from Lisp, enable the mode
if ARG is omitted or nil.
This mode causes morse code in the current channel to be unmorsed.
