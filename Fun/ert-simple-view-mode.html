ert-simple-view-mode is an interactive compiled Lisp function in `ert.el'.


(ert-simple-view-mode)

Parent mode: `special-mode'.

Major mode for viewing auxiliary information in ERT.

In addition to any hooks its parent mode `special-mode' might have run,
this mode runs the hook `ert-simple-view-mode-hook', as the final step
during initialization.
