eshell-mode is an interactive autoloaded compiled Lisp function in `esh-mode.el'.


(eshell-mode)

Emacs shell interactive mode.

This mode runs the hook `eshell-mode-hook', as the final step
during initialization.


Uses keymap `eshell-mode-map', which is not currently defined.
