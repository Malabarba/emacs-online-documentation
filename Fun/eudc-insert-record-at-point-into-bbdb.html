eudc-insert-record-at-point-into-bbdb is an interactive autoloaded Lisp function in `eudc-export.el'.

It is bound to <menu-bar> <tools> <directory-search> <Insert Record into BBDB>.

(eudc-insert-record-at-point-into-bbdb)

Insert record at point into the BBDB database.
This function can only be called from a directory query result buffer.
