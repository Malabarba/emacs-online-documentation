eww-bookmark-mode is an interactive compiled Lisp function in `eww.el'.


(eww-bookmark-mode)

Mode for listing bookmarks.

key binding
--- -------

C-k eww-bookmark-kill
RET eww-bookmark-browse
C-y eww-bookmark-yank
- negative-argument
0 .. 9 digit-argument
q quit-window
<remap> Prefix Command



This mode runs the hook `eww-bookmark-mode-hook', as the final step
during initialization.
