(eww-buffers-mode)

Mode for listing buffers.

key binding
--- -------

C-k eww-buffer-kill
RET eww-buffer-select
- negative-argument
0 .. 9 digit-argument
n eww-buffer-show-next
p eww-buffer-show-previous
q quit-window
<remap> Prefix Command



This mode runs the hook `eww-buffers-mode-hook', as the final step
during initialization.
