--- title: "expand-jump-to-previous-slot" layout: doc --- expand-jump-to-previous-slot is an interactive autoloaded compiled Lisp function in `expand.el'.

It is bound to C-x a p.

(expand-jump-to-previous-slot)

Move the cursor to the previous slot in the last abbrev expansion.
This is used only in conjunction with `expand-add-abbrevs'.
