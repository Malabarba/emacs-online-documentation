facemenu-remove-face-props is an interactive compiled Lisp function in `facemenu.el'.

It is bound to <C-down-mouse-2> <rm>, <menu-bar> <edit> <props> <rm>.

(facemenu-remove-face-props START END)

Remove `face' and `mouse-face' text properties.
