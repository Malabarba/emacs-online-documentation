--- title: "facemenu-remove-special" layout: doc --- facemenu-remove-special is an interactive compiled Lisp function in `facemenu.el'.

It is bound to <C-down-mouse-2> <sp> s, <menu-bar> <edit> <props> <sp> s.

(facemenu-remove-special START END)

Remove all the "special" text properties from the region.
These special properties include `invisible', `intangible' and `read-only'.
