facemenu-set-bold-italic is an interactive Lisp function.

It is bound to M-o l, <C-down-mouse-2> <fc> l, <menu-bar> <edit> <props> <fc> l.

(facemenu-set-bold-italic)

Select face `bold-italic' for subsequent insertion.
If the mark is active and there is no prefix argument,
apply face `bold-italic' to the region instead.
This command was defined by `facemenu-add-new-face'.
