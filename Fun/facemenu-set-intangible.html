--- title: "facemenu-set-intangible" layout: doc --- facemenu-set-intangible is an interactive compiled Lisp function in `facemenu.el'.

It is bound to <C-down-mouse-2> <sp> t, <menu-bar> <edit> <props> <sp> t.

(facemenu-set-intangible START END)

Make the region intangible: disallow moving into it.
This sets the `intangible' text property; it can be undone with
`facemenu-remove-special'.
