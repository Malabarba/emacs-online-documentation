filesets-file-open is a compiled Lisp function in `filesets.el'.

(filesets-file-open OPEN-FUNCTION FILE-NAME &optional FILESET-NAME)

Open FILE-NAME using OPEN-FUNCTION.
If OPEN-FUNCTION is nil, its value will be deduced from FILESET-NAME.
