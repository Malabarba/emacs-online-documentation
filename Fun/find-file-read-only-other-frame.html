find-file-read-only-other-frame is an interactive compiled Lisp function in `files.el'.

It is bound to C-x 5 r.

(find-file-read-only-other-frame FILENAME &optional WILDCARDS)

Edit file FILENAME in another frame but don't allow changes.
Like C-x 5 C-f, but marks buffer as read-only.
Use M-x toggle-read-only to permit editing.
