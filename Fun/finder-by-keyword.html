finder-by-keyword is an interactive autoloaded compiled Lisp function in `finder.el'.

It is bound to C-h p, <f1> p, <help> p, <menu-bar> <help-menu> <find-emacs-packages>.

(finder-by-keyword)

Find packages matching a given keyword.
