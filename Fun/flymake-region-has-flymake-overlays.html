--- title: "flymake-region-has-flymake-overlays" layout: doc --- flymake-region-has-flymake-overlays is a compiled Lisp function in `flymake.el'.

(flymake-region-has-flymake-overlays BEG END)

Check if region specified by BEG and END has overlay.
Return t if it has at least one flymake overlay, nil if no overlay.
