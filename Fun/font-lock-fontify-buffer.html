--- title: "font-lock-fontify-buffer" layout: doc --- font-lock-fontify-buffer is an interactive compiled Lisp function in `font-lock.el'.


(font-lock-fontify-buffer &optional INTERACTIVELY)

This function is for interactive use only;
in Lisp code use `font-lock-ensure' or `font-lock-flush' instead.

Fontify the current buffer the way the function `font-lock-mode' would.
