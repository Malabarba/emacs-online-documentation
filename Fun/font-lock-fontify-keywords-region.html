--- title: "font-lock-fontify-keywords-region" layout: doc --- font-lock-fontify-keywords-region is a compiled Lisp function in `font-lock.el'.

(font-lock-fontify-keywords-region START END &optional LOUDLY)

Fontify according to `font-lock-keywords' between START and END.
START should be at the beginning of a line.
LOUDLY, if non-nil, allows progress-meter bar.
