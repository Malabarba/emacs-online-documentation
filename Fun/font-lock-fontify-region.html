--- title: "font-lock-fontify-region" layout: doc --- font-lock-fontify-region is a compiled Lisp function in `font-lock.el'.

(font-lock-fontify-region BEG END &optional LOUDLY)

Fontify the text between BEG and END.
If LOUDLY is non-nil, print status messages while fontifying.
This works by calling `font-lock-fontify-region-function'.
