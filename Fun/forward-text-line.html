forward-text-line is an alias for `nroff-forward-text-line' in `nroff-mode.el'.

(forward-text-line &optional CNT)

This function is obsolete since 22.1;
use `nroff-forward-text-line' instead.

Go forward one nroff text line, skipping lines of nroff requests.
An argument is a repeat count; if negative, move backward.
