--- title: "frame-configuration-to-register" layout: doc --- frame-configuration-to-register is an interactive compiled Lisp function in `register.el'.


(frame-configuration-to-register REGISTER)

This function is obsolete since 24.4;
use `frameset-to-register' instead.

Store the window configuration of all frames in register REGISTER.
Use C-x r j to restore the configuration.
Argument is a character, naming the register.

Interactively, reads the register using `register-read-with-preview'.
