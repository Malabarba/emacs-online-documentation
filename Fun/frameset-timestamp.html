frameset-timestamp is a compiled Lisp function in `frameset.el'.

(frameset-timestamp FRAMESET)

This function has a compiler macro `frameset-timestamp--cmacro'.

Return the creation timestamp of FRAMESET.

The value is in the format returned by `current-time'.
This slot cannot be modified.
