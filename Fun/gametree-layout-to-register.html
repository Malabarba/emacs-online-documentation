--- title: "gametree-layout-to-register" layout: doc --- gametree-layout-to-register is an interactive compiled Lisp function in `gametree.el'.


(gametree-layout-to-register REGISTER)

Store current tree layout in register REGISTER.
Use M-x gametree-apply-register-layout to restore that configuration.
Argument is a character, naming the register.

Interactively, reads the register using `register-read-with-preview'.
