gdb-disassembly-mode is an interactive compiled Lisp function in `gdb-mi.el'.

(gdb-disassembly-mode)

Parent mode: `gdb-parent-mode'.

Major mode for GDB disassembly information.

In addition to any hooks its parent mode `gdb-parent-mode' might have run,
this mode runs the hook `gdb-disassembly-mode-hook', as the final step
during initialization.
