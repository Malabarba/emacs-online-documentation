gdb-memory-mode is an interactive compiled Lisp function in `gdb-mi.el'.

(gdb-memory-mode)

Parent mode: `gdb-parent-mode'.

Major mode for examining memory.

In addition to any hooks its parent mode `gdb-parent-mode' might have run,
this mode runs the hook `gdb-memory-mode-hook', as the final step
during initialization.
