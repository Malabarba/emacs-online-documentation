gdb-script-mode is an interactive autoloaded compiled Lisp function in `gud.el'.

(gdb-script-mode)

Parent mode: `prog-mode'.

Major mode for editing GDB scripts.

In addition to any hooks its parent mode `prog-mode' might have run,
this mode runs the hook `gdb-script-mode-hook', as the final step
during initialization.
