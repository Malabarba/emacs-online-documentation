(gdbmi-bnf-async-record)

Implementation of the following GDB/MI output grammar rules:

async-record ==>
exec-async-output | status-async-output | notify-async-output

exec-async-output ==>
[ token ] '*' async-output

status-async-output ==>
[ token ] '+' async-output

notify-async-output ==>
[ token ] '=' async-output

async-output ==>
async-class ( ',' result )* nl
