gdbmi-bnf-result-record is a compiled Lisp function in `gdb-mi.el'.

(gdbmi-bnf-result-record)

Implementation of the following GDB/MI output grammar rule:

result-record ==>
[ token ] '^' result-class ( ',' result )* nl

token ==>
any sequence of digits.
