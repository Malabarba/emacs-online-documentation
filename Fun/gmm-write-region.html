gmm-write-region is an autoloaded compiled Lisp function in `gmm-utils.el'.

(gmm-write-region START END FILENAME &optional APPEND VISIT LOCKNAME MUSTBENEW)

Compatibility function for `write-region'.

In XEmacs, the seventh argument of `write-region' specifies the
coding-system.
