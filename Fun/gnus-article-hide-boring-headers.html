--- title: "gnus-article-hide-boring-headers" layout: doc --- gnus-article-hide-boring-headers is an interactive autoloaded Lisp function in `gnus-art.el'.


(gnus-article-hide-boring-headers &optional INTERACTIVE &rest ARGS)

Toggle hiding of headers that aren't very interesting.
If given a negative prefix, always show; if given a positive prefix,
always hide.
