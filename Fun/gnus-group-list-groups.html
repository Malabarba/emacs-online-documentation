gnus-group-list-groups is an interactive autoloaded compiled Lisp function in `gnus-group.el'.

(gnus-group-list-groups &optional LEVEL UNREAD LOWEST)

List newsgroups with level LEVEL or lower that have unread articles.
Default is all subscribed groups.
If argument UNREAD is non-nil, groups with no unread articles are also
listed.

Also see the `gnus-group-use-permanent-levels' variable.
