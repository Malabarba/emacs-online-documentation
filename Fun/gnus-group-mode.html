gnus-group-mode is an interactive compiled Lisp function in `gnus-group.el'.

(gnus-group-mode)

Major mode for reading news.

All normal editing commands are switched off.

The group buffer lists (some of) the groups available. For instance,
`l' will list all subscribed groups with unread articles, while `A z'
lists all zombie groups.

Groups that are displayed can be entered with `SPC'. To subscribe
to a group not displayed, type `U'.

For more in-depth information on this mode, read the manual (`C-c TAB').
