--- title: "gnus-output-to-rmail" layout: doc --- gnus-output-to-rmail is a compiled Lisp function in `gnus-util.el'.

(gnus-output-to-rmail FILENAME &optional ASK)

Append the current article to an Rmail file named FILENAME.
In Emacs 22 this writes Babyl format; in Emacs 23 it writes mbox unless
FILENAME exists and is Babyl format.
