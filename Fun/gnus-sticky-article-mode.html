gnus-sticky-article-mode is an interactive compiled Lisp function in `gnus-art.el'.

(gnus-sticky-article-mode)

Parent mode: `gnus-article-mode'.

Mode for sticky articles.

In addition to any hooks its parent mode `gnus-article-mode' might have run,
this mode runs the hook `gnus-sticky-article-mode-hook', as the final step
during initialization.
