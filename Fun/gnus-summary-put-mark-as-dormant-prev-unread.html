gnus-summary-put-mark-as-dormant-prev-unread is an interactive Lisp function in `gnus-sum.el'.

(gnus-summary-put-mark-as-dormant-prev-unread N)

Mark the current article as dormantprevious unread.
If N, the prefix, then repeat N times.
If N is negative, move in reverse order.
The difference between N and the actual number of articles marked is
returned.
