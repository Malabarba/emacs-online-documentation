help-with-tutorial-spec-language is an interactive compiled Lisp function in `menu-bar.el'.

It is bound to <menu-bar> <help-menu> <emacs-tutorial-language-specific>.

(help-with-tutorial-spec-language)

Use the Emacs tutorial, specifying which language you want.
