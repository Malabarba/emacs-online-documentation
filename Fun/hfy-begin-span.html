hfy-begin-span is a compiled Lisp function in `htmlfontify.el'.

(hfy-begin-span STYLE TEXT-BLOCK TEXT-ID TEXT-BEGINS-BLOCK-P)

Default handler to begin a span of text.
Insert "<span class="STYLE" ...>". See
`hfy-begin-span-handler' for more information.
