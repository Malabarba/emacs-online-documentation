--- title: "hfy-colour-vals" layout: doc --- hfy-colour-vals is a compiled Lisp function in `htmlfontify.el'.

(hfy-colour-vals COLOUR)

Where COLOUR is a color name or #XXXXXX style triplet, return a
list of three (16 bit) rgb values for said color.

If a window system is unavailable, calls `hfy-fallback-colour-values'.
