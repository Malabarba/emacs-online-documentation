--- title: "hs-mouse-toggle-hiding" layout: doc --- hs-mouse-toggle-hiding is an interactive compiled Lisp function in `hideshow.el'.


(hs-mouse-toggle-hiding E)

Toggle hiding/showing of a block.
This command should be bound to a mouse key.
Argument E is a mouse event used by `mouse-set-point'.
See `hs-hide-block' and `hs-show-block'.
