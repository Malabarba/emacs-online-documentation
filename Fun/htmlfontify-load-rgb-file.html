--- title: "htmlfontify-load-rgb-file" layout: doc --- htmlfontify-load-rgb-file is an interactive autoloaded Lisp function in `hfy-cmap.el'.


(htmlfontify-load-rgb-file &optional FILE)

Load an X11 style rgb.txt FILE.
Search `hfy-rgb-load-path' if FILE is not specified.
Loads the variable `hfy-rgb-txt-colour-map', which is used by
`hfy-fallback-colour-values'.
