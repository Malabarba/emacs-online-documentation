icalendar-convert-diary-to-ical is an alias for `icalendar-export-file' in `icalendar.el'.

(icalendar-convert-diary-to-ical DIARY-FILENAME ICAL-FILENAME)

This function is obsolete since 22.1;
use `icalendar-export-file' instead.

Export diary file to iCalendar format.
All diary entries in the file DIARY-FILENAME are converted to iCalendar
format. The result is appended to the file ICAL-FILENAME.
