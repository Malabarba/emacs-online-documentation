--- title: "image-dired-mouse-display-image" layout: doc --- image-dired-mouse-display-image is an interactive compiled Lisp function in `image-dired.el'.


(image-dired-mouse-display-image EVENT)

Use mouse EVENT, call `image-dired-display-image' to display image.
Track this in associated dired buffer if `image-dired-track-movement' is
non-nil.
