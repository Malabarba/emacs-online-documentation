image-dired-setup-dired-keybindings is an interactive autoloaded compiled Lisp function in `image-dired.el'.

(image-dired-setup-dired-keybindings)

Setup easy-to-use keybindings for the commands to be used in dired mode.
Note that n, p and and will be hijacked and bound to
`image-dired-dired-x-line'.
