image-minor-mode is an interactive autoloaded compiled Lisp function in `image-mode.el'.


(image-minor-mode &optional ARG)

Toggle Image minor mode in this buffer.
With a prefix argument ARG, enable Image minor mode if ARG is
positive, and disable it otherwise. If called from Lisp, enable
the mode if ARG is omitted or nil.

Image minor mode provides the key C-c C-c,
to switch back to `image-mode' and display an image file as the
actual image.
