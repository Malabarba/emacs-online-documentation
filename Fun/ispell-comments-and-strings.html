ispell-comments-and-strings is an interactive autoloaded compiled Lisp function in `ispell.el'.

It is bound to <menu-bar> <tools> <spell> <ispell-comments-and-strings>.

(ispell-comments-and-strings)

Check comments and strings in the current buffer for spelling errors.
