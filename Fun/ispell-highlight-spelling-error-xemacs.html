--- title: "ispell-highlight-spelling-error-xemacs" layout: doc --- ispell-highlight-spelling-error-xemacs is a compiled Lisp function in `ispell.el'.

(ispell-highlight-spelling-error-xemacs START END &optional HIGHLIGHT)

Highlight the word from START to END using `isearch-highlight'.
When the optional third arg HIGHLIGHT is set, the word is highlighted,
otherwise it is displayed normally.
