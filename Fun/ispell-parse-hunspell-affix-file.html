ispell-parse-hunspell-affix-file is a compiled Lisp function in `ispell.el'.

(ispell-parse-hunspell-affix-file DICT-KEY)

Parse hunspell affix file to extract parameters for `DICT-KEY'.
Return a list in `ispell-dictionary-alist' format.
