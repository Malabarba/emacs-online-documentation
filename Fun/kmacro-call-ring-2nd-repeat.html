kmacro-call-ring-2nd-repeat is an interactive compiled Lisp function in `kmacro.el'.

It is bound to C-x C-k C-l.

(kmacro-call-ring-2nd-repeat ARG)

Execute second keyboard macro in macro ring.
This is like `kmacro-call-ring-2nd', but allows repeating macro commands
without repeating the prefix.
