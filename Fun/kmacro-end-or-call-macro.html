--- title: "kmacro-end-or-call-macro" layout: doc --- kmacro-end-or-call-macro is an interactive autoloaded compiled Lisp function in `kmacro.el'.

It is bound to <f4>.

(kmacro-end-or-call-macro ARG &optional NO-REPEAT)

End kbd macro if currently being defined; else call last kbd macro.
With numeric prefix ARG, repeat macro that many times.
With C-u, call second macro in macro ring.
