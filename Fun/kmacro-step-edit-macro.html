--- title: "kmacro-step-edit-macro" layout: doc --- kmacro-step-edit-macro is an interactive compiled Lisp function in `kmacro.el'.

It is bound to C-x C-k SPC.

(kmacro-step-edit-macro)

Step edit and execute last keyboard macro.

To customize possible responses, change the "bindings" in `kmacro-step-edit-map'.
