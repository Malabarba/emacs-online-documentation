--- title: "latex-string-prefix-p" layout: doc --- latex-string-prefix-p is an alias for `string-prefix-p' in `tex-mode.el'.

(latex-string-prefix-p PREFIX STRING &optional IGNORE-CASE)

This function is obsolete since 24.3;
use `string-prefix-p' instead.

Return non-nil if PREFIX is a prefix of STRING.
If IGNORE-CASE is non-nil, the comparison is done without paying attention
to case differences.
