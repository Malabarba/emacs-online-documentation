make-frame-command is an interactive compiled Lisp function in `frame.el'.

It is bound to C-x 5 2, <menu-bar> <file> <make-frame>.

(make-frame-command)

Make a new frame, on the same terminal as the selected frame.
If the terminal is a text-only terminal, this also selects the
new frame.
