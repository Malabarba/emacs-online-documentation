--- title: "makeinfo-recenter-compilation-buffer" layout: doc --- makeinfo-recenter-compilation-buffer is an interactive autoloaded compiled Lisp function in `makeinfo.el'.


(makeinfo-recenter-compilation-buffer LINENUM)

Redisplay `*compilation*' buffer so most recent output can be seen.
The last line of the buffer is displayed on
line LINE of the window, or centered if LINE is nil.
