mark-calendar-days-named is an alias for `calendar-mark-days-named' in `diary-lib.el'.

(mark-calendar-days-named DAYNAME &optional COLOR)

This function is obsolete since 23.1;
use `calendar-mark-days-named' instead.

Mark all dates in the calendar window that are day DAYNAME of the week.
0 means all Sundays, 1 means all Mondays, and so on.
Optional argument COLOR is passed to `calendar-mark-visible-date' as MARK.
