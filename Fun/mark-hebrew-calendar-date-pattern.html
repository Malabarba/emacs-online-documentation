mark-hebrew-calendar-date-pattern is an alias for `calendar-hebrew-mark-date-pattern' in `cal-hebrew.el'.

(mark-hebrew-calendar-date-pattern MONTH DAY YEAR &optional COLOR)

This function is obsolete since 23.1;
use `calendar-hebrew-mark-date-pattern' instead.

Mark dates in calendar window that conform to Hebrew date MONTH/DAY/YEAR.
A value of 0 in any position is a wildcard. Optional argument COLOR is
passed to `calendar-mark-visible-date' as MARK.
