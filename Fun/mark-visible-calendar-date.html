mark-visible-calendar-date is an alias for `calendar-mark-visible-date' in `calendar.el'.

(mark-visible-calendar-date DATE &optional MARK)

This function is obsolete since 23.1;
use `calendar-mark-visible-date' instead.

Mark DATE in the calendar window with MARK.
MARK is a single-character string, a list of face attributes/values, or a face.
MARK defaults to `diary-entry-marker'.
