menu-bar-kill-ring-save is an alias for `kill-ring-save' in `menu-bar.el'.


(menu-bar-kill-ring-save BEG END &optional REGION)

This function is obsolete since 24.1;
use `kill-ring-save' instead.

Save the region as if killed, but don't kill it.
In Transient Mark mode, deactivate the mark.
If `interprogram-cut-function' is non-nil, also save the text for a window
system cut and paste.

If you want to append the killed line to the last killed text,
use C-M-w before M-w.

The optional argument REGION if non-nil, indicates that we're not just copying
some text between BEG and END, but we're copying the region.

This command is similar to `copy-region-as-kill', except that it gives
visual feedback indicating the extent of the region being copied.
