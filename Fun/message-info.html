--- title: "message-info" layout: doc --- message-info is an interactive autoloaded compiled Lisp function in `message.el'.


(message-info &optional ARG)

Display the Message manual.

Prefixed with one C-u, display the Emacs MIME
manual. With two C-u's, display the EasyPG or
PGG manual, depending on the value of `mml2015-use'.
