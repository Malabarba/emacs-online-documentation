mh-delete-seq is an interactive autoloaded compiled Lisp function in `mh-seq.el'.


(mh-delete-seq SEQUENCE)

Delete SEQUENCE.

You are prompted for the sequence to delete. Note that this
deletes only the sequence, not the messages in the sequence. If
you want to delete the messages, use "C-u
M-x mh-delete-msg".
