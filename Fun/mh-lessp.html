mh-lessp is an autoloaded Lisp function in `mh-utils.el'.

(mh-lessp MSG1 MSG2)

Return the lesser of two message indicators MSG1 and MSG2.
Strings are "smaller" than numbers.
Valid values are things like "cur", "last", 1, and 1820.
