mh-letter-mode is an interactive autoloaded Lisp function in `mh-letter.el'.

(mh-letter-mode)

Parent mode: `mail-mode'.

Mode for composing letters in MH-E.

When you have finished composing, type C-c C-c to send
the message using the MH mail handling system.

There are two types of tags used by MH-E when composing MIME
messages: MML and MH. The option `mh-compose-insertion' controls
what type of tags are inserted by MH-E commands. These tags can
be converted to MIME body parts by running C-c C-e for
MH-style directives or C-c RET m for MML tags.

Options that control this mode can be changed with
M-x customize-group; specify the "mh-compose" group.

When a message is composed, the hooks `text-mode-hook',
`mail-mode-hook', and `mh-letter-mode-hook' are run (in that
order).
