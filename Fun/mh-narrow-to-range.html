--- title: "mh-narrow-to-range" layout: doc --- mh-narrow-to-range is an interactive autoloaded compiled Lisp function in `mh-limit.el'.


(mh-narrow-to-range RANGE)

Limit to RANGE.

Check the documentation of `mh-interactive-range' to see how
RANGE is read in interactive use.

Use / w to undo this command.
