--- title: "mixal-mode" layout: doc --- mixal-mode is an interactive autoloaded compiled Lisp function in `mixal-mode.el'.


(mixal-mode)

Parent mode: `prog-mode'.

Major mode for the mixal asm language.

In addition to any hooks its parent mode `prog-mode' might have run,
this mode runs the hook `mixal-mode-hook', as the final step
during initialization.
