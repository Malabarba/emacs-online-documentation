mml-secure-message-sign is an interactive compiled Lisp function in `mml-sec.el'.


(mml-secure-message-sign &optional METHOD)

Add MML tags to sign the entire message.
Use METHOD if given. Else use `mml-secure-method' or
`mml-default-sign-method'.
