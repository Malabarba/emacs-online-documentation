mouse-drag-mode-line is an interactive compiled Lisp function in `mouse.el'.

It is bound to <bottom-divider> <down-mouse-1>, <mode-line> <down-mouse-1>.

(mouse-drag-mode-line START-EVENT)

Change the height of a window by dragging on the mode line.
