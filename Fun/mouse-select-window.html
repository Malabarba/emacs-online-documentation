mouse-select-window is an interactive compiled Lisp function in `mouse.el'.

It is bound to <vertical-line> <mouse-1>, <header-line> <mouse-1>, <mode-line> <drag-mouse-1>, <mode-line> <mouse-1>.

(mouse-select-window CLICK)

Select the window clicked on; don't move point.
