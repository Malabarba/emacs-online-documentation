net-utils-mode is an interactive compiled Lisp function in `net-utils.el'.


(net-utils-mode)

Parent mode: `special-mode'.

Major mode for interacting with an external network utility.

In addition to any hooks its parent mode `special-mode' might have run,
this mode runs the hook `net-utils-mode-hook', as the final step
during initialization.
