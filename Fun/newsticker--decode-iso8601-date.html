newsticker--decode-iso8601-date is a compiled Lisp function in `newst-backend.el'.

(newsticker--decode-iso8601-date ISO8601-STRING)

Return ISO8601-STRING in format like `decode-time'.
Converts from ISO-8601 to Emacs representation.
Examples:
2004-09-17T05:09:49.001+00:00
2004-09-17T05:09:49+00:00
2004-09-17T05:09+00:00
2004-09-17T05:09:49
2004-09-17T05:09
2004-09-17
2004-09
2004
