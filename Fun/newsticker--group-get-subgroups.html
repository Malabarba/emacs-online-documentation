newsticker--group-get-subgroups is a compiled Lisp function in `newst-treeview.el'.

(newsticker--group-get-subgroups GROUP &optional RECURSIVE)

Return list of subgroups for GROUP.
If RECURSIVE is non-nil recursively get subgroups and return a nested list.
