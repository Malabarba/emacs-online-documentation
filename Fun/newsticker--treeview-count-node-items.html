newsticker--treeview-count-node-items is a compiled Lisp function in `newst-treeview.el'.

(newsticker--treeview-count-node-items FEED &optional ISVIRTUAL)

Count number of relevant items for a treeview node.
FEED gives the name of the feed or group. If ISVIRTUAL is non-nil
the feed is a virtual feed.
