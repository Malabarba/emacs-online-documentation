newsticker--treeview-propertize-tag is a compiled Lisp function in `newst-treeview.el'.

(newsticker--treeview-propertize-tag TAG &optional NUM-NEW NT-ID FEED VFEED)

Return propertized copy of string TAG.
Optional argument NUM-NEW is used for choosing face, other
arguments NT-ID, FEED, and VFEED are added as properties.
