--- title: "newsticker-next-item-same-feed" layout: doc --- newsticker-next-item-same-feed is an interactive compiled Lisp function in `newst-plainview.el'.


(newsticker-next-item-same-feed)

Go to next news item in the same feed.
Return new buffer position. If no item is found below point or if
auto-narrow-to-item is enabled, nil is returned.
