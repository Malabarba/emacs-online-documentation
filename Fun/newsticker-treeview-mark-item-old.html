newsticker-treeview-mark-item-old is an interactive compiled Lisp function in `newst-treeview.el'.


(newsticker-treeview-mark-item-old &optional DONT-PROCEED)

Mark current item as old unless it is obsolete.
Move to next item unless DONT-PROCEED is non-nil.
