nnmail-fix-eudora-headers is an alias for `nnmail-ignore-broken-references' in `nnmail.el'.

(nnmail-fix-eudora-headers)

This function is obsolete since Emacs 23.1;
use `nnmail-ignore-broken-references' instead.

Ignore the References line and use In-Reply-To

Eudora has a broken References line, but an OK In-Reply-To.
