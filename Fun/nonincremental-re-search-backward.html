nonincremental-re-search-backward is an interactive compiled Lisp function in `menu-bar.el'.

It is bound to <menu-bar> <edit> <search> <re-search-backward>.

(nonincremental-re-search-backward STRING)

Read a regular expression and search backward for it nonincrementally.
