nslookup-mode is an interactive compiled Lisp function in `net-utils.el'.


(nslookup-mode)

Parent mode: `comint-mode'.

Major mode for interacting with the nslookup program.

In addition to any hooks its parent mode `comint-mode' might have run,
this mode runs the hook `nslookup-mode-hook', as the final step
during initialization.
