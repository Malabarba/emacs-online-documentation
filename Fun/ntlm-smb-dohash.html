ntlm-smb-dohash is a compiled Lisp function in `ntlm.el'.

(ntlm-smb-dohash IN KEY FORW)

Return the hash value for a string IN and a string KEY.
Length of IN and KEY are 64. FORW non-nil means forward, nil means
backward.
