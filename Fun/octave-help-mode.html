octave-help-mode is an interactive compiled Lisp function in `octave.el'.


(octave-help-mode)

Parent mode: `help-mode'.

Major mode for displaying Octave documentation.

In addition to any hooks its parent mode `help-mode' might have run,
this mode runs the hook `octave-help-mode-hook', as the final step
during initialization.
