org-display-inline-modification-hook is an alias for `org-display-inline-remove-overlay' in `org.el'.

(org-display-inline-modification-hook OV AFTER BEG END &optional LEN)

This function is obsolete since 24.3;
use `org-display-inline-remove-overlay' instead.

Remove inline-display overlay if a corresponding region is modified.
