org-minutes-to-hh:mm-string is an alias for `org-minutes-to-clocksum-string' in `org.el'.

(org-minutes-to-hh:mm-string M)

This function is obsolete since Org mode version 8.0;
use `org-minutes-to-clocksum-string' instead.

Format number of minutes as a clocksum string.
The format is determined by `org-time-clocksum-format',
`org-time-clocksum-use-fractional' and
`org-time-clocksum-fractional-format' and
`org-time-clocksum-use-effort-durations'.
