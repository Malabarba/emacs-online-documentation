--- title: "org-tags-sparse-tree" layout: doc --- org-tags-sparse-tree is an alias for `org-match-sparse-tree' in `org.el'.


(org-tags-sparse-tree &optional TODO-ONLY MATCH)

Create a sparse tree according to tags string MATCH.
MATCH can contain positive and negative selection of tags, like
"+WORK+URGENT-WITHBOSS".
If optional argument TODO-ONLY is non-nil, only select lines that are
also TODO lines.
