--- title: "perl-electric-terminator" layout: doc --- perl-electric-terminator is an interactive compiled Lisp function in `perl-mode.el'.


(perl-electric-terminator ARG)

This function is obsolete since 24.4;
use `electric-indent-mode' instead.

Insert character and maybe adjust indentation.
If at end-of-line, and not in a comment or a quote, correct the indentation.
