ps-run-mode is an interactive compiled Lisp function in `ps-mode.el'.


(ps-run-mode)

Parent mode: `comint-mode'.

Major mode in interactive PostScript window.
This mode is invoked from `ps-mode' and should not be called directly.

In addition to any hooks its parent mode `comint-mode' might have run,
this mode runs the hook `ps-run-mode-hook', as the final step
during initialization.
