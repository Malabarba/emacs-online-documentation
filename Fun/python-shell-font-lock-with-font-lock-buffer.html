--- title: "python-shell-font-lock-with-font-lock-buffer" layout: doc --- python-shell-font-lock-with-font-lock-buffer is a Lisp macro in `python.el'.

(python-shell-font-lock-with-font-lock-buffer &rest BODY)

Execute the forms in BODY in the font-lock buffer.
The value returned is the value of the last form in BODY. See
also `with-current-buffer'.
