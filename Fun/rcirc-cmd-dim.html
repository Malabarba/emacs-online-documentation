--- title: "rcirc-cmd-dim" layout: doc --- rcirc-cmd-dim is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-dim NICK &optional PROCESS TARGET)

Manage the dim nick list.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
