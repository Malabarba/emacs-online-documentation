rcirc-cmd-invite is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-invite NICK-CHANNEL &optional PROCESS TARGET)

Invite NICK to CHANNEL.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
