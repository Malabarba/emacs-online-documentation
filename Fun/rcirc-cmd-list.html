--- title: "rcirc-cmd-list" layout: doc --- rcirc-cmd-list is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-list CHANNELS &optional PROCESS TARGET)

Request information on CHANNELS from server.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
