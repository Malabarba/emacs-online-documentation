--- title: "rcirc-cmd-msg" layout: doc --- rcirc-cmd-msg is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-msg MESSAGE &optional PROCESS TARGET)

Send private MESSAGE to TARGET.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
