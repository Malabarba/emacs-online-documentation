--- title: "rcirc-cmd-oper" layout: doc --- rcirc-cmd-oper is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-oper ARGS &optional PROCESS TARGET)

Send operator command to server.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
