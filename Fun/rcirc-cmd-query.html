--- title: "rcirc-cmd-query" layout: doc --- rcirc-cmd-query is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-query NICK &optional PROCESS TARGET)

Open a private chat buffer to NICK.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
