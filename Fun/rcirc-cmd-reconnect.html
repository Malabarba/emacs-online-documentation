--- title: "rcirc-cmd-reconnect" layout: doc --- rcirc-cmd-reconnect is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-reconnect ## &optional PROCESS TARGET)

Reconnect to current server.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
