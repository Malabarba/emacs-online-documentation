--- title: "rcirc-cmd-topic" layout: doc --- rcirc-cmd-topic is an interactive compiled Lisp function in `rcirc.el'.


(rcirc-cmd-topic TOPIC &optional PROCESS TARGET)

List TOPIC for the TARGET channel.
With a prefix arg, prompt for new topic.

Note: If PROCESS or TARGET are nil, the values given
by `rcirc-buffer-process' and `rcirc-target' will be used.
