rcirc-debug is a compiled Lisp function in `rcirc.el'.

(rcirc-debug PROCESS TEXT)

Add an entry to the debug log including PROCESS and TEXT.
Debug text is written to `rcirc-debug-buffer' if `rcirc-debug-flag'
is non-nil.
