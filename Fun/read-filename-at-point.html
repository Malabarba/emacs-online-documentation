read-filename-at-point is an alias for `dired-x-read-filename-at-point' in `dired-x.el'.

(read-filename-at-point PROMPT)

This function is obsolete since 24.1;
use `dired-x-read-filename-at-point' instead.

Return filename prompting with PROMPT with completion.
If `current-prefix-arg' is non-nil, uses name at point as guess.
