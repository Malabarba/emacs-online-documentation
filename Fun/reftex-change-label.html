--- title: "reftex-change-label" layout: doc --- reftex-change-label is an interactive autoloaded compiled Lisp function in `reftex-global.el'.


(reftex-change-label &optional FROM TO)

Run `query-replace-regexp' of FROM with TO in all macro arguments.
Works on the entire multifile document.
If you exit (C-g, RET or q), you can resume the query replace
with the command M-x tags-loop-continue.
No active TAGS table is required.
