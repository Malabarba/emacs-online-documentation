--- title: "rfc2231-encode-string" layout: doc --- rfc2231-encode-string is a compiled Lisp function in `rfc2231.el'.

(rfc2231-encode-string PARAM VALUE)

Return and PARAM=VALUE string encoded according to RFC2231.
Use `mml-insert-parameter' or `mml-insert-parameter-string' to insert
the result of this function.
