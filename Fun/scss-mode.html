--- title: "scss-mode" layout: doc --- scss-mode is an interactive autoloaded compiled Lisp function in `css-mode.el'.


(scss-mode)

Parent mode: `css-mode'.

Major mode to edit "Sassy CSS" files.

In addition to any hooks its parent mode `css-mode' might have run,
this mode runs the hook `scss-mode-hook', as the final step
during initialization.
