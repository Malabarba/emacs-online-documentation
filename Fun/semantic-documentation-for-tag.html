--- title: "semantic-documentation-for-tag" layout: doc --- semantic-documentation-for-tag is an autoloaded Lisp function in `doc.el'.

(semantic-documentation-for-tag &optional TAG NOSNARF)

Find documentation from TAG and return it as a clean string.
TAG might have DOCUMENTATION set in it already. If not, there may be
some documentation in a comment preceding TAG's definition which we
can look for. When appropriate, this can be overridden by a language specific
enhancement.
Optional argument NOSNARF means to only return the lexical analyzer token for it.
If nosnarf if 'lex, then only return the lex token.
