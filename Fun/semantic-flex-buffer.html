semantic-flex-buffer is an alias for `semantic-lex-buffer' in `lex.el'.

(semantic-flex-buffer &optional DEPTH)

This function is obsolete since 23.2;
use `semantic-lex-buffer' instead.

Lex the current buffer.
Optional argument DEPTH is the depth to scan into lists.
