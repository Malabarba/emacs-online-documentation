semantic-flex-keywords is an alias for `semantic-lex-keywords' in `lex.el'.

(semantic-flex-keywords &optional PROPERTY)

This function is obsolete since 23.2;
use `semantic-lex-keywords' instead.

Return a list of lexical keywords.
If optional PROPERTY is non-nil, return only keywords which have a
PROPERTY set.
