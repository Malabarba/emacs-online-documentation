semantic-flex-map-keywords is an alias for `semantic-lex-map-keywords' in `lex.el'.

(semantic-flex-map-keywords FUN &optional PROPERTY)

This function is obsolete since 23.2;
use `semantic-lex-map-keywords' instead.

Call function FUN on every lexical keyword.
If optional PROPERTY is non-nil, call FUN only on every keyword which
as a PROPERTY value. FUN receives a lexical keyword as argument.
