--- title: "semantic-symref-find-tags-by-name" layout: doc --- semantic-symref-find-tags-by-name is an interactive autoloaded Lisp function in `symref.el'.


(semantic-symref-find-tags-by-name NAME &optional SCOPE)

Find a list of tags by NAME in the current project.
Optional SCOPE specifies which file set to search. Defaults to 'project.
Refers to `semantic-symref-tool', to determine the reference tool to use
for the current buffer.
Returns an object of class `semantic-symref-result'.
