set-buffer-process-coding-system is an interactive compiled Lisp function in `mule.el'.

It is bound to C-x RET p, <menu-bar> <options> <mule> <set-various-coding-system> <set-buffer-process-coding-system>.

(set-buffer-process-coding-system DECODING ENCODING)

Set coding systems for the process associated with the current buffer.
DECODING is the coding system to be used to decode input from the process,
ENCODING is the coding system to be used to encode output to the process.

For a list of possible coding systems, use M-x list-coding-systems.
