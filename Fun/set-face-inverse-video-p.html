set-face-inverse-video-p is an alias for `set-face-inverse-video' in `faces.el'.


(set-face-inverse-video-p FACE INVERSE-VIDEO-P &optional FRAME)

This function is obsolete since 24.4;
use `set-face-inverse-video' instead.

Specify whether face FACE is in inverse video.
INVERSE-VIDEO-P non-nil means FACE displays explicitly in inverse video.
INVERSE-VIDEO-P nil means FACE explicitly is not in inverse video.
FRAME nil or not specified means change face on all frames.
Use `set-face-attribute' to ``unspecify'' the inverse video attribute.
