set-frame-width is a built-in function in `C source code'.

(set-frame-width FRAME WIDTH &optional PRETEND PIXELWISE)

Specify that the frame FRAME has WIDTH columns.
Optional third arg PRETEND non-nil means that redisplay should use WIDTH
columns but that the idea of the actual width of the frame should not
be changed. Optional fourth argument PIXELWISE non-nil means that FRAME
should be WIDTH pixels wide.
