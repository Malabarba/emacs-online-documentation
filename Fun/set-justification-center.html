set-justification-center is an interactive compiled Lisp function in `fill.el'.

It is bound to <C-down-mouse-2> <ju> c, <menu-bar> <edit> <props> <ju> c.

(set-justification-center B E)

Make paragraphs in the region centered.
If the mark is not active, this applies to the current paragraph.
