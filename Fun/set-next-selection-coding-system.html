set-next-selection-coding-system is an interactive compiled Lisp function in `mule.el'.

It is bound to C-x RET X, <menu-bar> <options> <mule> <set-various-coding-system> <set-next-selection-coding-system>.

(set-next-selection-coding-system CODING-SYSTEM)

Use CODING-SYSTEM for next communication with other window system clients.
This setting is effective for the next communication only.
