set-selection-coding-system is an interactive compiled Lisp function in `mule.el'.

It is bound to C-x RET x, <menu-bar> <options> <mule> <set-various-coding-system> <set-selection-coding-system>.

(set-selection-coding-system CODING-SYSTEM)

Make CODING-SYSTEM used for communicating with other X clients.
When sending or receiving text via cut_buffer, selection, and clipboard,
the text is encoded or decoded by CODING-SYSTEM.
