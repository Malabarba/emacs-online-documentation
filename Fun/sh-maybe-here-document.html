--- title: "sh-maybe-here-document" layout: doc --- sh-maybe-here-document is an interactive compiled Lisp function in `sh-script.el'.


(sh-maybe-here-document ARG)

This function is obsolete since 24.3;
use `sh-electric-here-document-mode' instead.

Insert self. Without prefix, following unquoted `<' inserts here document.
The document is bounded by `sh-here-document-word'.
