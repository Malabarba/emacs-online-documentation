--- title: "simula-submit-bug-report" layout: doc --- simula-submit-bug-report is an alias for `report-emacs-bug' in `simula.el'.


(simula-submit-bug-report TOPIC &optional UNUSED)

This function is obsolete since 24.4;
use `report-emacs-bug' instead.

Report a bug in GNU Emacs.
Prompts for bug subject. Leaves you in a mail buffer.
