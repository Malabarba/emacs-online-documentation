size-indication-mode is an interactive compiled Lisp function in `simple.el'.

It is bound to <menu-bar> <options> <showhide> <size-indication-mode>.

(size-indication-mode &optional ARG)

Toggle buffer size display in the mode line (Size Indication mode).
With a prefix argument ARG, enable Size Indication mode if ARG is
positive, and disable it otherwise.

If called from Lisp, enable the mode if ARG is omitted or nil.
