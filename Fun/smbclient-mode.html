smbclient-mode is an interactive compiled Lisp function in `net-utils.el'.

(smbclient-mode)

Parent mode: `comint-mode'.

Major mode for interacting with the smbclient program.

In addition to any hooks its parent mode `comint-mode' might have run,
this mode runs the hook `smbclient-mode-hook', as the final step
during initialization.
