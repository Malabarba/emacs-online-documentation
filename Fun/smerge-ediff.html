smerge-ediff is an interactive autoloaded compiled Lisp function in `smerge-mode.el'.


(smerge-ediff &optional NAME-MINE NAME-OTHER NAME-BASE)

Invoke ediff to resolve the conflicts.
NAME-MINE, NAME-OTHER, and NAME-BASE, if non-nil, are used for the
buffer names.
