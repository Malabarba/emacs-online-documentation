smime-mode is an interactive compiled Lisp function in `smime.el'.

(smime-mode)

Major mode for browsing, viewing and fetching certificates.

All normal editing commands are switched off.


The following commands are available:

key binding
--- -------

- negative-argument
0 .. 9 digit-argument
f smime-certificate-info
q smime-exit
Prefix Command



This mode runs the hook `smime-mode-hook', as the final step
during initialization.
