speedbar-buffers-line-path is an alias for `speedbar-buffers-line-directory' in `speedbar.el'.

(speedbar-buffers-line-path &optional DEPTH)

This function is obsolete since 22.1;
use `speedbar-buffers-line-directory' instead.

Fetch the directory of the file (buffer) specified on the current line.
Optional argument DEPTH specifies the current depth of the back search.
