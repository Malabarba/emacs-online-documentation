speedbar-maybee-jump-to-attached-frame is an alias for `dframe-maybee-jump-to-attached-frame' in `speedbar.el'.

(speedbar-maybee-jump-to-attached-frame)

This function is obsolete since 24.4;
use `dframe-maybee-jump-to-attached-frame' instead.

Jump to the attached frame ONLY if this was not a mouse event.
