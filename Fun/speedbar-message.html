speedbar-message is an alias for `dframe-message' in `speedbar.el'.

(speedbar-message FMT &rest ARGS)

This function is obsolete since 24.4;
use `dframe-message' instead.

Like `message', but for use in a dedicated frame.
Argument FMT is the format string, and ARGS are the arguments for message.
