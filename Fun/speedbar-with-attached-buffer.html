--- title: "speedbar-with-attached-buffer" layout: doc --- speedbar-with-attached-buffer is an alias for `dframe-with-attached-buffer' in `speedbar.el'.

(speedbar-with-attached-buffer &rest FORMS)

This function is obsolete since 24.4;
use `dframe-with-attached-buffer' instead.

Execute FORMS in the attached frame's special buffer.
Optionally select that frame if necessary.
