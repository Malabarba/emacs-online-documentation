srecode-semantic-handle-:texitag is an autoloaded Lisp function in `texi.el'.

(srecode-semantic-handle-:texitag DICT)

Add macros into the dictionary DICT based on the current :tag file.
Adds the following:
TAGDOC - Texinfo formatted doc string for :tag.
