strokes-report-bug is an alias for `report-emacs-bug' in `strokes.el'.


(strokes-report-bug TOPIC &optional UNUSED)

This function is obsolete since 24.1;
use `report-emacs-bug' instead.

Report a bug in GNU Emacs.
Prompts for bug subject. Leaves you in a mail buffer.
