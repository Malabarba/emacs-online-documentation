tetris-mode is an interactive compiled Lisp function in `tetris.el'.

(tetris-mode)

A mode for playing Tetris.

This mode runs the hook `tetris-mode-hook', as the final step
during initialization.

key binding

SPC tetris-move-bottom
n tetris-start-game
p tetris-pause-game
