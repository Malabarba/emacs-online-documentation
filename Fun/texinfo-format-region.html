texinfo-format-region is an interactive autoloaded compiled Lisp function in `texinfmt.el'.


(texinfo-format-region REGION-BEGINNING REGION-END)

Convert the current region of the Texinfo file to Info format.
This lets you see what that part of the file will look like in Info.
The command is bound to M-x texinfo-format-region. The text that is
converted to Info is stored in a temporary buffer.
