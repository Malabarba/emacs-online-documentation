texinfo-tex-view is an interactive compiled Lisp function in `texinfo.el'.


(texinfo-tex-view)

View `.dvi' file made by M-x texinfo-tex-region or M-x texinfo-tex-buffer.
This runs the shell command defined by `tex-dvi-view-command'.
