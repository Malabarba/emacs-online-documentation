texinfo-texindex is an interactive compiled Lisp function in `texinfo.el'.


(texinfo-texindex)

Run `texindex' on unsorted index files.
The index files are made by M-x texinfo-tex-region or M-x texinfo-tex-buffer.
This runs the shell command defined by `texinfo-texindex-command'.
