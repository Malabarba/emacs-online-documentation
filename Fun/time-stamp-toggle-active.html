time-stamp-toggle-active is an interactive autoloaded compiled Lisp function in `time-stamp.el'.


(time-stamp-toggle-active &optional ARG)

Toggle `time-stamp-active', setting whether M-x time-stamp updates a buffer.
With ARG, turn time stamping on if and only if arg is positive.
