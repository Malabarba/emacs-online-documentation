--- title: "timeclock-update-modeline" layout: doc --- timeclock-update-modeline is an alias for `timeclock-update-mode-line' in `timeclock.el'.


(timeclock-update-modeline)

This function is obsolete since 24.3;
use `timeclock-update-mode-line' instead.

Update the `timeclock-mode-string' displayed in the mode line.
The value of `timeclock-relative' affects the display as described in
that variable's documentation.
