This function no longer exists in emacs 24.4

todo-print is an interactive autoloaded compiled Lisp function in `todo-mode.el'.

(todo-print &optional CATEGORY-PR-PAGE)

Print todo summary using `todo-print-function'.
If CATEGORY-PR-PAGE is non-nil, a page separator `^L' is inserted
between each category.

Number of entries for each category is given by `todo-print-priorities'.
