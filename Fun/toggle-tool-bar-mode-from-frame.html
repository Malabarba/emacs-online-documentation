toggle-tool-bar-mode-from-frame is an interactive autoloaded compiled Lisp function in `tool-bar.el'.

It is bound to <menu-bar> <options> <showhide> <showhide-tool-bar>.

(toggle-tool-bar-mode-from-frame &optional ARG)

Toggle tool bar on or off, based on the status of the current frame.
See `tool-bar-mode' for more information.
