toggle-uniquify-buffer-names is an interactive compiled Lisp function in `menu-bar.el'.

It is bound to <menu-bar> <options> <uniquify>.

(toggle-uniquify-buffer-names &optional INTERACTIVELY)

Toggle whether to uniquify buffer names by adding parent directory names.
In an interactive call, record this option as a candidate for saving
by "Save Options" in Custom buffers.
