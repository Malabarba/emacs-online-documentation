--- title: "toggle-use-system-font" layout: doc --- toggle-use-system-font is an interactive compiled Lisp function in `menu-bar.el'.

It is bound to <menu-bar> <options> <menu-system-font>.

(toggle-use-system-font &optional INTERACTIVELY)

Toggle whether to use the monospaced font defined by the system.
In an interactive call, record this option as a candidate for saving
by "Save Options" in Custom buffers.
