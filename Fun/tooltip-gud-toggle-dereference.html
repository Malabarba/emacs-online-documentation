tooltip-gud-toggle-dereference is an alias for `gud-tooltip-dereference' in `gud.el'.

(tooltip-gud-toggle-dereference &optional ARG)

This function is obsolete since 22.1;
use `gud-tooltip-dereference' instead.

Toggle whether tooltips should show `* expr' or `expr'.
With arg, dereference expr if ARG is positive, otherwise do not dereference.
