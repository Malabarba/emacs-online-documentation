--- title: "tramp-compat-split-string" layout: doc --- tramp-compat-split-string is a compiled Lisp function in `tramp-compat.el'.

(tramp-compat-split-string STRING PATTERN)

Like `split-string' but omit empty strings.
In Emacs, (split-string "/foo/bar" "/") returns ("foo" "bar").
This is, the first, empty, element is omitted. In XEmacs, the first
element is not omitted.
