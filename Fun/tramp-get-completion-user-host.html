tramp-get-completion-user-host is a compiled Lisp function in `tramp.el'.

(tramp-get-completion-user-host METHOD PARTIAL-USER PARTIAL-HOST USER HOST)

Returns the most expanded string for user and host name completion.
PARTIAL-USER must match USER, PARTIAL-HOST must match HOST.
