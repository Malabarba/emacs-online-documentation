--- title: "tramp-gvfs-file-name-handler" layout: doc --- tramp-gvfs-file-name-handler is an autoloaded Lisp function in `tramp-gvfs.el'.

(tramp-gvfs-file-name-handler OPERATION &rest ARGS)

Invoke the GVFS related OPERATION.
First arg specifies the OPERATION, second arg is a list of arguments to
pass to the OPERATION.
