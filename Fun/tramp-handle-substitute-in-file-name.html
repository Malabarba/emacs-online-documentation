tramp-handle-substitute-in-file-name is a compiled Lisp function in `tramp.el'.

(tramp-handle-substitute-in-file-name FILENAME)

Like `substitute-in-file-name' for Tramp files.
"//" and "/~" substitute only in the local filename part.
If the URL Tramp syntax is chosen, "//" as method delimiter and "/~" at
beginning of local filename are not substituted.
