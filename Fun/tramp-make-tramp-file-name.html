--- title: "tramp-make-tramp-file-name" layout: doc --- tramp-make-tramp-file-name is a compiled Lisp function in `tramp.el'.

(tramp-make-tramp-file-name METHOD USER HOST LOCALNAME &optional HOP)

Constructs a Tramp file name from METHOD, USER, HOST and LOCALNAME.
When not nil, an optional HOP is prepended.
