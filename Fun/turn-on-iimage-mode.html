turn-on-iimage-mode is an alias for `iimage-mode' in `iimage.el'.


(turn-on-iimage-mode &optional ARG)

This function is obsolete since 24.1;
use `iimage-mode' instead.

Toggle Iimage mode on or off.
With a prefix argument ARG, enable Iimage mode if ARG is
positive, and disable it otherwise. If called from Lisp, enable
the mode if ARG is omitted or nil, and toggle it if ARG is `toggle'.
