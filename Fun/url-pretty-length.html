--- title: "url-pretty-length" layout: doc --- url-pretty-length is an alias for `file-size-human-readable' in `url-util.el'.

(url-pretty-length FILE-SIZE &optional FLAVOR)

This function is obsolete since 24.4;
use `file-size-human-readable' instead.

Produce a string showing FILE-SIZE in human-readable form.

Optional second argument FLAVOR controls the units and the display format:

If FLAVOR is nil or omitted, each kilobyte is 1024 bytes and the produced
suffixes are "k", "M", "G", "T", etc.
If FLAVOR is `si', each kilobyte is 1000 bytes and the produced suffixes
are "k", "M", "G", "T", etc.
If FLAVOR is `iec', each kilobyte is 1024 bytes and the produced suffixes
are "KiB", "MiB", "GiB", "TiB", etc.
