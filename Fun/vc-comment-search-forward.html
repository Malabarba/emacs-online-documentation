vc-comment-search-forward is an alias for `log-edit-comment-search-forward' in `log-edit.el'.


(vc-comment-search-forward STR)

This function is obsolete since 22.1;
use `log-edit-comment-search-forward' instead.

Search forwards through comment history for a substring match of STR.
