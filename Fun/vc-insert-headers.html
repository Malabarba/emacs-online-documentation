vc-insert-headers is an interactive autoloaded compiled Lisp function in `vc.el'.

It is bound to C-x v h, <menu-bar> <tools> <vc> <vc-insert-header>.

(vc-insert-headers)

Insert headers into a file for use with a version control system.
Headers desired are inserted at point, and are pulled from
the variable `vc-BACKEND-header'.
