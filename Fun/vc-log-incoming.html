vc-log-incoming is an interactive autoloaded compiled Lisp function in `vc.el'.

It is bound to C-x v I, <menu-bar> <tools> <vc> <vc-log-in>.

(vc-log-incoming &optional REMOTE-LOCATION)

Show a log of changes that will be received with a pull operation from REMOTE-LOCATION.
When called interactively with a prefix argument, prompt for REMOTE-LOCATION.
