--- title: "vc-previous-comment" layout: doc --- vc-previous-comment is an alias for `log-edit-previous-comment' in `log-edit.el'.


(vc-previous-comment ARG)

This function is obsolete since 22.1;
use `log-edit-previous-comment' instead.

Cycle backwards through comment history.
With a numeric prefix ARG, go back ARG comments.
