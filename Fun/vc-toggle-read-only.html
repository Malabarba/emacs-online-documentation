--- title: "vc-toggle-read-only" layout: doc --- vc-toggle-read-only is an alias for `toggle-read-only' in `vc-hooks.el'.


(vc-toggle-read-only &optional ARG INTERACTIVE)

This function is obsolete since 24.1;
use `read-only-mode' instead (or `toggle-read-only' in older versions of Emacs).

Change whether this buffer is read-only.
