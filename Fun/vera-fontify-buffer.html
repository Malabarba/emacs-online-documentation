vera-fontify-buffer is an alias for `font-lock-fontify-buffer' in `vera-mode.el'.


(vera-fontify-buffer &optional INTERACTIVELY)

This function is obsolete since 25.1;
use `font-lock-fontify-buffer' instead.

Fontify the current buffer the way the function `font-lock-mode' would.
