verilog-delete-trailing-whitespace is a compiled Lisp function in `verilog-mode.el'.

(verilog-delete-trailing-whitespace)

Delete trailing spaces or tabs, but not newlines nor linefeeds.
Also add missing final newline.

To call this from the command line, see M-x verilog-batch-diff-auto.

To call on M-x verilog-auto, set `verilog-auto-delete-trailing-whitespace'.
