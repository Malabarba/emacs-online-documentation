--- title: "vhdl-electric-mode" layout: doc --- vhdl-electric-mode is an interactive compiled Lisp function in `vhdl-mode.el'.


(vhdl-electric-mode &optional ARG)

Toggle VHDL electric mode.
With a prefix argument ARG, enable the mode if ARG is positive,
and disable it otherwise. If called from Lisp, enable it if ARG
is omitted or nil.
