vhdl-speedbar-insert-project-hierarchy is a compiled Lisp function in `vhdl-mode.el'.

(vhdl-speedbar-insert-project-hierarchy PROJECT INDENT &optional RESCAN)

Insert hierarchy of PROJECT. Rescan directories if RESCAN is non-nil,
otherwise use cached data.
