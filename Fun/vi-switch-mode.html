vi-switch-mode is an interactive compiled Lisp function in `vi.el'.

It is bound to C-x ~.

(vi-switch-mode ARG MODE-CHAR)

Switch the major mode of current buffer as specified by the following char key binding
--- -------

O outline-mode
P picture-mode
a abbrev-mode
c c-mode
d vi-debugging
e emacs-lisp-mode
f auto-fill-mode
g prolog-mode
h hanoi
i info-mode
l lisp-mode
n nroff-mode
o overwrite-mode
r vi-readonly-mode
t text-mode
v vi-mode
x tex-mode
~ vi-back-to-old-mode
