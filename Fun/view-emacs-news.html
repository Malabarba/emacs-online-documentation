view-emacs-news is an interactive compiled Lisp function in `help.el'.

It is bound to C-h n, C-h C-n, <f1> n, <f1> C-n, <help> n, <help> C-n, <menu-bar> <help-menu> <emacs-news>.

(view-emacs-news &optional VERSION)

Display info on recent changes to Emacs.
With argument, display info only for the selected version.
