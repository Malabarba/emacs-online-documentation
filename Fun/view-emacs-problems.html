--- title: "view-emacs-problems" layout: doc --- view-emacs-problems is an interactive compiled Lisp function in `help.el'.

It is bound to C-h C-p, <f1> C-p, <help> C-p, <menu-bar> <help-menu> <emacs-known-problems>.

(view-emacs-problems)

Display info on known problems with Emacs and possible workarounds.
