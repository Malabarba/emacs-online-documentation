--- title: "viper-mouse-catch-frame-switch" layout: doc --- viper-mouse-catch-frame-switch is an interactive compiled Lisp function in `viper-mous.el'.

It is bound to <M-S-down-mouse-2>, <M-S-down-mouse-1>.

(viper-mouse-catch-frame-switch EVENT ARG)

Catch the event of switching frame.
Usually is bound to a `down-mouse' event to work properly. See sample
bindings in the Viper manual.
