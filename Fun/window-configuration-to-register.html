--- title: "window-configuration-to-register" layout: doc --- window-configuration-to-register is an interactive compiled Lisp function in `register.el'.

It is bound to C-x r w.

(window-configuration-to-register REGISTER)

Store the window configuration of the selected frame in register REGISTER.
Use C-x r j to restore the configuration.
Argument is a character, naming the register.

Interactively, reads the register using `register-read-with-preview'.
