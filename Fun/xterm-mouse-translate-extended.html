xterm-mouse-translate-extended is a compiled Lisp function in `xt-mouse.el'.

(xterm-mouse-translate-extended EVENT)

Read a click and release event from XTerm.
Similar to `xterm-mouse-translate', but using the "1006"
extension, which supports coordinates >= 231 (see
http://invisible-island.net/xterm/ctlseqs/ctlseqs.html).
