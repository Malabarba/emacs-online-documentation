--- title: "2C-beyond-fill-column" layout: doc --- 2C-beyond-fill-column is a variable defined in `two-column.el'.
Its value is 
4


Documentation:
Base for calculating `fill-column' for a buffer in two-column minor mode.
The value of `fill-column' becomes `2C-window-width' for this buffer
minus this value.

You can customize this variable.
