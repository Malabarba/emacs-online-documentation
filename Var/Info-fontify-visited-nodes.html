--- title: "Info-fontify-visited-nodes" layout: doc --- Info-fontify-visited-nodes is a variable defined in `info.el'.
Its value is 
t


Documentation:
Non-nil to fontify references to visited nodes in `info-xref-visited' face.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
