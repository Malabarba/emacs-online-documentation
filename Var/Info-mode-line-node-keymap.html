Info-mode-line-node-keymap is a variable defined in `info.el'.
Its value is 


(keymap
 (mode-line keymap
	    (mouse-3 . Info-mouse-scroll-down)
	    (mouse-1 . Info-mouse-scroll-up)))


Documentation:
Keymap to put on the Info node name in the mode line.
