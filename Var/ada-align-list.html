ada-align-list is a variable defined in `ada-mode.el'.
Its value is 


(("[^:]\\(\\s-*\\):[^:]" 1 t)
 ("[^=]\\(\\s-+\\)=[^=]" 1 t)
 ("\\(\\s-*\\)use\\s-" 1)
 ("\\(\\s-*\\)--" 1))


Documentation:
Ada support for align.el <= 2.2.
This variable provides regular expressions on which to align different lines.
See `align-mode-alist' for more information.
