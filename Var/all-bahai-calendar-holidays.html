all-bahai-calendar-holidays is a variable defined in `calendar.el'.
Its value is 
nil

  • This variable is an alias for `calendar-bahai-all-holidays-flag'.
  • This variable is obsolete since 23.1;
    use `calendar-bahai-all-holidays-flag' instead.

Documentation:
If nil, show only major holidays from the Bahá'í calendar.
These are the days on which work and school must be suspended.
Otherwise, show all the holidays that would appear in a complete Bahá'í
calendar.

You can customize this variable.
