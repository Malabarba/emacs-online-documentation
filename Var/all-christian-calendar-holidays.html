all-christian-calendar-holidays is a variable defined in `calendar.el'.
Its value is nil

This variable is an alias for `calendar-christian-all-holidays-flag'.
This variable is obsolete since 23.1;
use `calendar-christian-all-holidays-flag' instead.

Documentation:
If nil, show only major holidays from the Christian calendar.
This means only those Christian holidays that appear on secular calendars.
Otherwise, show all the holidays that would appear in a complete Christian
calendar.

You can customize this variable.
