all-hebrew-calendar-holidays is a variable defined in `calendar.el'.
Its value is 
nil

  • This variable is an alias for `calendar-hebrew-all-holidays-flag'.
  • This variable is obsolete since 23.1;
    use `calendar-hebrew-all-holidays-flag' instead.

Documentation:
If nil, show only major holidays from the Hebrew calendar.
This means only those Jewish holidays that appear on secular calendars.
Otherwise, show all the holidays that would appear in a complete Hebrew
calendar.

You can customize this variable.
