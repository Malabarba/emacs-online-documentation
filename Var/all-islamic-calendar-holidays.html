all-islamic-calendar-holidays is a variable defined in `calendar.el'.
Its value is nil

This variable is an alias for `calendar-islamic-all-holidays-flag'.
This variable is obsolete since 23.1;
use `calendar-islamic-all-holidays-flag' instead.

Documentation:
If nil, show only major holidays from the Islamic calendar.
This means only those Islamic holidays that appear on secular calendars.
Otherwise, show all the holidays that would appear in a complete Islamic
calendar.

You can customize this variable.
