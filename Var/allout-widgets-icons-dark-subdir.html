--- title: "allout-widgets-icons-dark-subdir" layout: doc --- allout-widgets-icons-dark-subdir is a variable defined in `allout-widgets.el'.
Its value is 
"icons/allout-widgets/dark-bg/"


Documentation:
Directory on `image-load-path' holding allout icons for dark backgrounds.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.1 of Emacs.
