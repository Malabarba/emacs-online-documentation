ange-ftp-auto-save is a variable defined in `ange-ftp.el'.
Its value is 
0


Documentation:
If 1, allow ange-ftp files to be auto-saved.
If 0, inhibit auto-saving of ange-ftp files.
Don't use any other value.

You can customize this variable.
