--- title: "ange-ftp-bs2000-short-filename-regexp" layout: doc --- ange-ftp-bs2000-short-filename-regexp is a variable defined in `ange-ftp.el'.
Its value is 
"[-A-Z0-9$#@.]*[A-Z][-A-Z0-9$#@.]*"

  • This variable may be risky if used as a file-local variable.

Documentation:
Regular expression to match for a valid short BS2000 file name.
