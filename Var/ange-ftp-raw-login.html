--- title: "ange-ftp-raw-login" layout: doc --- ange-ftp-raw-login is a variable defined in `ange-ftp.el'.
Its value is 
nil


Documentation:
Use raw FTP commands for login, if account password is not nil.
Some FTP implementations need this, e.g. ftp in NT 4.0.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 21.3 of Emacs.
