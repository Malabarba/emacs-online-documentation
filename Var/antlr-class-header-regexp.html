antlr-class-header-regexp is a variable defined in `antlr-mode.el'.
Its value is 


"\\(class\\)[ 	]+\\([A-Za-z\300-\326\330-\337]\\sw*\\)[ 	]+\\(extends\\)[ 	]+\\([A-Za-z\300-\326\330-\337]\\sw*\\)[ 	]*;"


Documentation:
Regexp matching class headers.
