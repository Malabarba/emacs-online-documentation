antlr-tool-command is a variable defined in `antlr-mode.el'.
Its value is "java antlr.Tool"

This variable may be risky if used as a file-local variable.

Documentation:
Command used in M-x antlr-run-tool to run the Antlr tool.
This variable should include all options passed to Antlr except the
option "-glib" which is automatically suggested if necessary.

You can customize this variable.
