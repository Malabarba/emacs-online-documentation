--- title: "archive-visit-single-files" layout: doc --- archive-visit-single-files is a variable defined in `arc-mode.el'.
Its value is 
nil


Documentation:
If non-nil, opening an archive with a single file visits that file.
If nil, visiting such an archive displays the archive summary.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
