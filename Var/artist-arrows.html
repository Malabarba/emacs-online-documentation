Documentation:
A vector of characters to use as arrows.

The vector is 8 elements long and contains a character for each
direction, or nil if there is no suitable character to use for arrow
in that direction.

The directions are as follows:

5 6 7
\ | /
4 - * - 0
/ | \
3 2 1
