--- title: "bibtex-reference-key" layout: doc --- bibtex-reference-key is a variable defined in `bibtex.el'.
Its value is 
"[][[:alnum:].:;?!`'/*@+|()<>&_^$-]+"

  • This variable may be risky if used as a file-local variable.

Documentation:
Regexp matching the reference key part of a BibTeX entry.
