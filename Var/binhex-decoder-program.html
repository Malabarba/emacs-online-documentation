--- title: "binhex-decoder-program" layout: doc --- binhex-decoder-program is a variable defined in `binhex.el'.
Its value is 
"hexbin"

  • This variable may be risky if used as a file-local variable.

Documentation:
Non-nil value should be a string that names a binhex decoder.
The program should expect to read binhex data on its standard
input and write the converted data to its standard output.

You can customize this variable.
