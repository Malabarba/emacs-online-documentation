blink-cursor is a variable defined in `frame.el'.
Its value is t

This variable is an alias for `blink-cursor-mode'.
This variable is obsolete since 22.1;
use `blink-cursor-mode' instead.

Documentation:
Non-nil if Blink-Cursor mode is enabled.
See the command `blink-cursor-mode' for a description of this minor mode.
Setting this variable directly does not take effect;
either customize it (see the info node `Easy Customization')
or call the function `blink-cursor-mode'.

You can customize this variable.
