calc-infinite-mode is a variable defined in `calc.el'.
Its value is 
nil


Documentation:
If nil, 1 / 0 is left unsimplified.
If 0, 1 / 0 is changed to inf (zeros are considered positive).
Otherwise, 1 / 0 is changed to uinf (undirected infinity).
