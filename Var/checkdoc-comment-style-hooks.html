checkdoc-comment-style-hooks is a variable defined in `checkdoc.el'.
Its value is nil

This variable is an alias for `checkdoc-comment-style-functions'.
This variable is obsolete since 24.3;
use `checkdoc-comment-style-functions' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Hook run after the standard comment style check is completed.
Must return nil if no errors are found, or a string describing the
problem discovered. This is useful for adding additional checks.
