compilation-always-kill is a variable defined in `compile.el'.
Its value is 
nil


Documentation:
If t, always kill a running compilation process before starting a new one.
If nil, ask to kill it.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
