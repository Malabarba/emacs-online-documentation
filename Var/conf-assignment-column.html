--- title: "conf-assignment-column" layout: doc --- conf-assignment-column is a variable defined in `conf-mode.el'.
Its value is 
24


Documentation:
Align assignments to this column by default with M-x conf-align-assignments.
If this number is negative, the `=' comes before the whitespace. Use 0 to
not align (only setting space according to `conf-assignment-space').

You can customize this variable.
