copyright-at-end-flag is a variable defined in `copyright.el'.
Its value is 
nil

  • This variable is safe as a file local variable if its value
    satisfies the predicate `booleanp'.

Documentation:
Non-nil means to search backwards from the end of the buffer for copyright.
This is useful for ChangeLogs.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
