cperl-have-help-regexp is a variable defined in `cperl-mode.el'.
Its value is
"[$@%*&][0-9a-zA-Z_:]+\\([ ]*[[{]\\)?\\|[$@]\\^[a-zA-Z]\\|[$@][^ \n ]\\|-[a-zA-Z]\\|\\\\[a-zA-Z0]\\|^=[a-z][a-zA-Z0-9_]*\\|[-!&*+,-./<=>?\\\\^|~]+\\|[a-zA-Z_0-9:]+\\|x=\\|#!"

Documentation:
Matches places in the buffer we can find help for.
