crisp-mode-modeline-string is a variable defined in `crisp.el'.
Its value is " *CRiSP*"

This variable is an alias for `crisp-mode-mode-line-string'.
This variable is obsolete since 24.3;
use `crisp-mode-mode-line-string' instead.

Documentation:
String to display in the mode line when CRiSP emulation mode is enabled.

You can customize this variable.
