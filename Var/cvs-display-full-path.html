cvs-display-full-path is a variable defined in `pcvs-info.el'.
Its value is t

This variable is an alias for `cvs-display-full-name'.
This variable is obsolete since 22.1;
use `cvs-display-full-name' instead.

Documentation:
Specifies how the filenames should be displayed in the listing.
If non-nil, their full filename name will be displayed, else only the
non-directory part.

You can customize this variable.
