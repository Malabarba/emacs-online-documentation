--- title: "dabbrev-ignored-buffer-names" layout: doc --- dabbrev-ignored-buffer-names is a variable defined in `dabbrev.el'.
Its value is 
("*Messages*" "*Buffer List*")


Documentation:
List of buffer names that dabbrev should not check.
See also `dabbrev-ignored-buffer-regexps'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20.3 of Emacs.
