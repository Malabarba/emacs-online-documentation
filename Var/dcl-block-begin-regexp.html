dcl-block-begin-regexp is a variable defined in `dcl-mode.el'.
Its value is 
"loop[0-9]*:"


Documentation:
Regexp describing a command that begins an indented block in DCL.
Set to nil to only indent at THEN-ELSE-ENDIF.

You can customize this variable.
