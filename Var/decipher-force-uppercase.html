decipher-force-uppercase is a variable defined in `decipher.el'.
Its value is 
t


Documentation:
Non-nil means to convert ciphertext to uppercase.
nil means the case of the ciphertext is preserved.
This variable must be set before typing `M-x decipher'.

You can customize this variable.
