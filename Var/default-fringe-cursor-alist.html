default-fringe-cursor-alist is a variable defined in `C source code'.
Its value is ((box . filled-rectangle)
(hollow . hollow-rectangle)
(bar . vertical-bar)
(hbar . horizontal-bar)
(hollow-small . hollow-square))


This variable is obsolete since 23.2;
use `fringe-cursor-alist' instead.

Documentation:
Default value of `fringe-cursor-alist' for buffers that don't override it.
This is the same as (default-value 'fringe-cursor-alist').
