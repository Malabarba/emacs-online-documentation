desktop-enable is a variable defined in `desktop.el'.
Its value is 
nil

  • This variable is an alias for `desktop-save-mode'.
  • This variable is obsolete since 22.1;
    use `desktop-save-mode' instead.

Documentation:
Non-nil if Desktop-Save mode is enabled.
See the command `desktop-save-mode' for a description of this minor mode.
Setting this variable directly does not take effect;
either customize it (see the info node `Easy Customization')
or call the function `desktop-save-mode'.
