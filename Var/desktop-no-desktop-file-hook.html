--- title: "desktop-no-desktop-file-hook" layout: doc --- desktop-no-desktop-file-hook is a variable defined in `desktop.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Normal hook run when `desktop-read' can't find a desktop file.
Run in the directory in which the desktop file was sought.
May be used to show a dired buffer.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
