--- title: "desktop-restore-eager" layout: doc --- desktop-restore-eager is a variable defined in `desktop.el'.
Its value is 
t


Documentation:
Number of buffers to restore immediately.
Remaining buffers are restored lazily (when Emacs is idle).
If value is t, all buffers are restored immediately.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
