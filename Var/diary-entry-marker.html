diary-entry-marker is a variable defined in `calendar.el'.
Its value is 
diary


Documentation:
How to mark dates that have diary entries.
The value can be either a single-character string (e.g. "+") or a face.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
