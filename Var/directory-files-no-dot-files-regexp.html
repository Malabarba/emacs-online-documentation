--- title: "directory-files-no-dot-files-regexp" layout: doc --- directory-files-no-dot-files-regexp is a variable defined in `files.el'.
Its value is 
"^\\([^.]\\|\\.\\([^.]\\|\\..\\)\\).*"

  • This variable may be risky if used as a file-local variable.

Documentation:
Regexp matching any file name except "." and "..".
