dired-at-point-require-prefix is a variable defined in `ffap.el'.
Its value is 
nil


Documentation:
If non-nil, reverse the prefix argument to `dired-at-point'.
This is nil so neophytes notice ffap. Experts may prefer to
disable ffap most of the time.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20.3 of Emacs.
