dired-x-hands-off-my-keys is a variable defined in `dired-x.el'.
Its value is t

Documentation:
Non-nil means don't remap `find-file' to `dired-x-find-file'.
Similarly for `find-file-other-window' and `dired-x-find-file-other-window'.
If you change this variable without using M-x customize after `dired-x.el'
is loaded then call M-x dired-x-bind-find-file.
