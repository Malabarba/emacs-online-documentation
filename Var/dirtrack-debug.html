dirtrack-debug is a variable defined in `dirtrack.el'.
Its value is nil

Automatically becomes buffer-local when set.
This variable is an alias for `dirtrack-debug-mode'.
This variable is obsolete since 23.1;
use `dirtrack-debug-mode' instead.

Documentation:
Non-nil if Dirtrack-Debug mode is enabled.
Use the command `dirtrack-debug-mode' to change this variable.
