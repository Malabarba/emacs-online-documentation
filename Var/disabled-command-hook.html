disabled-command-hook is a variable defined in `novice.el'.
Its value is disabled-command-function

This variable is an alias for `disabled-command-function'.
This variable is obsolete since 22.1;
use `disabled-command-function' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Function to call to handle disabled commands.
If nil, the feature is disabled, i.e., all commands work normally.
