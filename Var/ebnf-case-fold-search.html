ebnf-case-fold-search is a variable defined in `ebnf2ps.el'.
Its value is 
nil


Documentation:
Non-nil means ignore case on matching.

It's only used when `ebnf-terminal-regexp' is non-nil and when `ebnf-syntax' is
`ebnf'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
