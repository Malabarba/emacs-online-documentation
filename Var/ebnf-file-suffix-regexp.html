--- title: "ebnf-file-suffix-regexp" layout: doc --- ebnf-file-suffix-regexp is a variable defined in `ebnf2ps.el'.
Its value is 
".[Bb][Nn][Ff]$"


Documentation:
Specify file name suffix that contains EBNF.

See `ebnf-eps-directory' command.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
