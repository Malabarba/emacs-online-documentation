--- title: "ebnf-horizontal-max-height" layout: doc --- ebnf-horizontal-max-height is a variable defined in `ebnf2ps.el'.
Its value is 
nil


Documentation:
Non-nil means to use maximum production height in horizontal orientation.

It is only used when `ebnf-horizontal-orientation' is non-nil.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
