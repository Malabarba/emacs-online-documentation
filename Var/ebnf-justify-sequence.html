ebnf-justify-sequence is a variable defined in `ebnf2ps.el'.
Its value is 
center


Documentation:
Specify justification of terms in a sequence inside alternatives.

Valid values are:

`left' left justification
`right' right justification
any other value centralize

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
