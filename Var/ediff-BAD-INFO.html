ediff-BAD-INFO is a variable defined in `ediff-init.el'.
Its value is 


"\n*** The Info file for Ediff, a part of the standard distribution\n*** of Emacs, does not seem to be properly installed.\n***\n*** Please contact your system administrator. "

  • This variable may be risky if used as a file-local variable.
