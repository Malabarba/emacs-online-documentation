--- title: "ediff-no-emacs-help-in-control-buffer" layout: doc --- ediff-no-emacs-help-in-control-buffer is a variable defined in `ediff-init.el'.
Its value is 
nil


Documentation:
Non-nil means C-h should not invoke Emacs help in control buffer.
Instead, C-h would jump to previous difference.

You can customize this variable.
