--- title: "ediff-prepare-buffer-hook" layout: doc --- ediff-prepare-buffer-hook is a variable defined in `ediff-init.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hooks run after buffers A, B, and C are set up.
For each buffer, the hooks are run with that buffer made current.

You can customize this variable.
