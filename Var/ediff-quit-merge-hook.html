--- title: "ediff-quit-merge-hook" layout: doc --- ediff-quit-merge-hook is a variable defined in `ediff-merg.el'.
Its value is 
ediff-maybe-save-and-delete-merge

  • This variable may be risky if used as a file-local variable.

Documentation:
Hooks to run before quitting a merge job.
The most common use is to save and delete the merge buffer.

You can customize this variable.
