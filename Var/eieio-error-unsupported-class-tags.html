--- title: "eieio-error-unsupported-class-tags" layout: doc --- eieio-error-unsupported-class-tags is a variable defined in `eieio-core.el'.
Its value is 
nil


Documentation:
Non-nil to throw an error if an encountered tag is unsupported.
This may prevent classes from CLOS applications from being used with EIEIO
since EIEIO does not support all CLOS tags.
