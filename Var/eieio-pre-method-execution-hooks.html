eieio-pre-method-execution-hooks is a variable defined in `eieio-core.el'.
Its value is 
nil

  • This variable is an alias for `eieio-pre-method-execution-functions'.
  • This variable is obsolete since 24.3;
    use `eieio-pre-method-execution-functions' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
Abnormal hook run just before an EIEIO method is executed.
The hook function must accept one argument, the list of forms
about to be executed.
