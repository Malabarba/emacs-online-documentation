electric-indent-functions-without-reindent is a variable defined in `electric.el'.
Its value is 


(indent-relative indent-to-left-margin indent-relative-maybe py-indent-line coffee-indent-line org-indent-line yaml-indent-line haskell-indentation-indent-line haskell-indent-cycle haskell-simple-indent yaml-indent-line)


Documentation:
List of indent functions that can't reindent.
If `line-indent-function' is one of those, then `electric-indent-mode' will
not try to reindent lines. It is normally better to make the major
mode set `electric-indent-inhibit', but this can be used as a workaround.
