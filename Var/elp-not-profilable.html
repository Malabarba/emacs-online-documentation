elp-not-profilable is a variable defined in `elp.el'.
Its value is
(elp-wrapper called-interactively-p error call-interactively apply current-time + byte-code-function-p functionp byte-code subrp indirect-function fboundp)


Documentation:
List of functions that cannot be profiled.
Those functions are used internally by the profiling code and profiling
them would thus lead to infinite recursion.
