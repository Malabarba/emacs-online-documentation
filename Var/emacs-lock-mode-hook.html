--- title: "emacs-lock-mode-hook" layout: doc --- emacs-lock-mode-hook is a variable defined in `emacs-lock.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering or leaving `emacs-lock-mode'.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)
