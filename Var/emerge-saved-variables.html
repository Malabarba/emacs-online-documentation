emerge-saved-variables is a variable defined in `emerge.el'.
Its value is 


((buffer-modified-p set-buffer-modified-p)
 buffer-read-only buffer-auto-save-file-name)

  • This variable may be risky if used as a file-local variable.

Documentation:
Variables and properties of a buffer which are saved, modified and restored
during a merge.
