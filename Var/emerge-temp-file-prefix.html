--- title: "emerge-temp-file-prefix" layout: doc --- emerge-temp-file-prefix is a variable defined in `emerge.el'.
Its value is 
"/tmp/emerge"

  • This variable is obsolete since 24.4;
    customize `temporary-file-directory' instead.

Documentation:
Prefix to put on Emerge temporary file names.
Do not start with `~/' or `~USERNAME/'.

You can customize this variable.
