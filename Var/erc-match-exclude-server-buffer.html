--- title: "erc-match-exclude-server-buffer" layout: doc --- erc-match-exclude-server-buffer is a variable defined in `erc-match.el'.
Its value is 
nil


Documentation:
If true, don't perform match on the server buffer; this is
useful for excluding all the things like MOTDs from the server
and other miscellaneous functions.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
