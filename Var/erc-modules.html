(pcomplete netsplit fill button match track completion readonly networks ring autojoin noncommands irccontrols move-to-prompt stamp menu list)

Original value was 
(netsplit fill button match track completion readonly networks ring autojoin noncommands irccontrols move-to-prompt stamp menu list)


Documentation:
A list of modules which ERC should enable.
If you set the value of this without using `customize' remember to call
(erc-update-modules) after you change it. When using `customize', modules
removed from the list will be disabled.

You can customize this variable.
