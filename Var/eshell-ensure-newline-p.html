eshell-ensure-newline-p is a variable defined in `esh-cmd.el'.
Its value is 
nil


Documentation:
If non-nil, ensure that a newline is emitted after a Lisp form.
This can be changed by Lisp forms that are evaluated from the Eshell
command line.
