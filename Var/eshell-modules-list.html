eshell-modules-list is a variable defined in `esh-module.el'.
Its value is
(eshell-alias eshell-banner eshell-basic eshell-cmpl eshell-dirs eshell-glob eshell-hist eshell-ls eshell-pred eshell-prompt eshell-script eshell-term eshell-unix)


Documentation:
A list of optional add-on modules to be loaded by Eshell.
Changes will only take effect in future Eshell buffers.

You can customize this variable.
