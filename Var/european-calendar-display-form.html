european-calendar-display-form is a variable defined in `calendar.el'.
Its value is ((if dayname
(concat dayname ", "))
day " " monthname " " year)


This variable is an alias for `calendar-european-date-display-form'.
This variable is obsolete since 23.1;
use `calendar-european-date-display-form' instead.
This variable may be risky if used as a file-local variable.

Documentation:
