european-date-diary-pattern is a variable defined in `calendar.el'.
Its value is ((day "/" month "[^/0-9]")
(day "/" month "/" year "[^0-9]")
(backup day " *" monthname "\\W+\\<\\([^*0-9]\\|\\([0-9]+[:aApP]\\)\\)")
(day " *" monthname " *" year "[^0-9]")
(dayname "\\W"))


This variable is an alias for `diary-european-date-forms'.
This variable is obsolete since 23.1;
use `diary-european-date-forms' instead.
This variable may be risky if used as a file-local variable.

Documentation:
List of pseudo-patterns describing the European style of dates.
The defaults are: DAY/MONTH; DAY/MONTH/YEAR; DAY MONTHNAME;
DAY MONTHNAME YEAR; DAYNAME. Normally you should not customize this, but
`diary-date-forms' (which see).

You can customize this variable.
