fancy-diary-buffer is a variable defined in `calendar.el'.
Its value is 
"*Fancy Diary Entries*"

  • This variable is an alias for `diary-fancy-buffer'.
  • This variable is obsolete since 23.1;
    use `diary-fancy-buffer' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
Name of the buffer used for the optional fancy display of the diary.
