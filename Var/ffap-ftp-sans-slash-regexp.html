ffap-ftp-sans-slash-regexp is a variable defined in `ffap.el'.
Its value is 


"\\`\\([^:@]+@[^:@]+:\\|[^@.:]+\\.[^@:]+:\\|[^:]+:[~/]\\)\\([^:]\\|\\'\\)"


Documentation:
Strings matching this are coerced to FTP file names by ffap.
That is, ffap just prepends "/". Set to nil to disable.
