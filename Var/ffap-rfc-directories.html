--- title: "ffap-rfc-directories" layout: doc --- ffap-rfc-directories is a variable defined in `ffap.el'.
Its value is 
nil


Documentation:
A list of directories to look for RFC files.
If a given RFC isn't in these then `ffap-rfc-path' is offered.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
