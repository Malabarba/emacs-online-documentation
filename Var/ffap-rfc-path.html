ffap-rfc-path is a variable defined in `ffap.el'.
Its value is 
"/anonymous@ftp.rfc-editor.org:/in-notes/rfc%s.txt"


Documentation:
A `format' string making a filename for RFC documents.
This can be an ange-ftp or Tramp remote filename to download, or
a local filename if you have full set of RFCs locally. See also
`ffap-rfc-directories'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
