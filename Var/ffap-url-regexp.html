--- title: "ffap-url-regexp" layout: doc --- ffap-url-regexp is a variable defined in `ffap.el'.
Its value is 


"\\(news\\(post\\)?:\\|mailto:\\|file:\\|\\(ftp\\|https?\\|telnet\\|gopher\\|www\\|wais\\)://\\)"


Documentation:
Regexp matching the beginning of a URI, for ffap.
If the value is nil, disable URL-matching features in ffap.
