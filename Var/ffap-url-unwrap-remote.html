--- title: "ffap-url-unwrap-remote" layout: doc --- ffap-url-unwrap-remote is a variable defined in `ffap.el'.
Its value is 
("ftp")


Documentation:
If non-nil, convert URLs to remote file names before prompting.
If the value is a list of strings, that specifies a list of URL
schemes (e.g. "ftp"); in that case, only convert those URLs.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
