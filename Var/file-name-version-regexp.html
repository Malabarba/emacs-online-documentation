file-name-version-regexp is a variable defined in `files.el'.
Its value is 


"\\(?:~\\|\\.~[-[:alnum:]:#@^._]+\\(?:~[[:digit:]]+\\)?~\\)"


Documentation:
Regular expression matching the backup/version part of a file name.
Used by `file-name-sans-versions'.
