find-file-hook is a variable defined in `files.el'.
Its value is
(mode-local-post-major-mode-change global-font-lock-mode-check-buffers epa-file-find-file-hook vc-find-file-hook cvs-insert-visited-file which-func-ff-hook)

Original value was nil

This variable may be risky if used as a file-local variable.

Documentation:
List of functions to be called after a buffer is loaded from a file.
The buffer's local variables (if any) will have been processed before the
functions are called.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 22.1 of Emacs.
