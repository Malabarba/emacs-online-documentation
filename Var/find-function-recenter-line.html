--- title: "find-function-recenter-line" layout: doc --- find-function-recenter-line is a variable defined in `find-func.el'.
Its value is 
1


Documentation:
The window line-number from which to start displaying a symbol definition.
A value of nil implies center the beginning of the definition.
See `find-function' and `find-variable'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20.3 of Emacs.
