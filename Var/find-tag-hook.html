--- title: "find-tag-hook" layout: doc --- find-tag-hook is a variable defined in `etags.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook to be run by M-x find-tag after finding a tag. See `run-hooks'.
The value in the buffer in which M-x find-tag is done is used,
not the value in the buffer M-x find-tag goes to.

You can customize this variable.
