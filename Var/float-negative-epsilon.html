float-negative-epsilon is a variable defined in `cl.el'.
Its value is nil

This variable is an alias for `cl-float-negative-epsilon'.
This variable may be risky if used as a file-local variable.

Documentation:
The smallest positive float that subtracts from 1.0 to give a distinct value.
For IEEE machines, it is about 1.11e-16.
Call `cl-float-limits' to set this.
