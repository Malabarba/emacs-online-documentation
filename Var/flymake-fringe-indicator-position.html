--- title: "flymake-fringe-indicator-position" layout: doc --- flymake-fringe-indicator-position is a variable defined in `flymake.el'.
Its value is 
left-fringe


Documentation:
The position to put flymake fringe indicator.
The value can be nil (do not use indicators), `left-fringe' or `right-fringe'.
See `flymake-error-bitmap' and `flymake-warning-bitmap'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
