flyspell-default-delayed-commands is a variable defined in `flyspell.el'.
Its value is 


(self-insert-command delete-backward-char backward-or-forward-delete-char delete-char scrollbar-vertical-drag backward-delete-char-untabify)

  • This variable may be risky if used as a file-local variable.

Documentation:
The standard list of delayed commands for Flyspell.
See `flyspell-delayed-commands'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 21.1 of Emacs.
