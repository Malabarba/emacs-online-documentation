flyspell-default-deplacement-commands is a variable defined in `flyspell.el'.
Its value is 


(next-line previous-line handle-switch-frame handle-select-window scroll-up scroll-down)

  • This variable may be risky if used as a file-local variable.

Documentation:
The standard list of deplacement commands for Flyspell.
See variable `flyspell-deplacement-commands'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 21.1 of Emacs.
