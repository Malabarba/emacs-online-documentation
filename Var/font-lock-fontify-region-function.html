--- title: "font-lock-fontify-region-function" layout: doc --- font-lock-fontify-region-function is a variable defined in `font-lock.el'.
Its value is 
font-lock-default-fontify-region

  • This variable may be risky if used as a file-local variable.

Documentation:
Function to use for fontifying a region.
It should take two args, the beginning and end of the region, and an optional
third arg VERBOSE. If VERBOSE is non-nil, the function should print status
messages. This is normally set via `font-lock-defaults'.
