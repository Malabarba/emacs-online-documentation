--- title: "gdb-control-commands-regexp" layout: doc --- gdb-control-commands-regexp is a variable defined in `gdb-mi.el'.
Its value is 


"^\\(commands\\|if\\|while\\|define\\|document\\|python\\|while-stepping\\|stepping\\|ws\\|actions\\)\\([[:blank:]]+.*\\)?$"


Documentation:
Regexp matching GDB commands that enter a recursive reading loop.
As long as GDB is in the recursive reading loop, it does not expect
commands to be prefixed by "-interpreter-exec console".
