gdb-many-windows is a variable defined in `gdb-mi.el'.
Its value is 
nil


Documentation:
Non-nil if Gdb-Many-Windows mode is enabled.
See the command `gdb-many-windows' for a description of this minor mode.
Setting this variable directly does not take effect;
either customize it (see the info node `Easy Customization')
or call the function `gdb-many-windows'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
