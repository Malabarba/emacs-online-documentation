--- title: "gdb-show-main" layout: doc --- gdb-show-main is a variable defined in `gdb-mi.el'.
Its value is 
nil


Documentation:
Non-nil means display source file containing the main routine at startup.
Also display the main routine in the disassembly buffer if present.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
