--- title: "global-subword-mode-hook" layout: doc --- global-subword-mode-hook is a variable defined in `subword.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering or leaving `global-subword-mode'.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)
