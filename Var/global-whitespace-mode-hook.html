--- title: "global-whitespace-mode-hook" layout: doc --- global-whitespace-mode-hook is a variable defined in `whitespace.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering or leaving `global-whitespace-mode'.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)
