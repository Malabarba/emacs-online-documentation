--- title: "gnus-confirm-treat-mail-like-news" layout: doc --- gnus-confirm-treat-mail-like-news is a variable defined in `gnus-msg.el'.
Its value is 
nil


Documentation:
If non-nil, Gnus will treat mail like news with regard to confirmation
when replying by mail. See the `gnus-confirm-mail-reply-to-news' variable
for fine-tuning this.
If nil, Gnus will never ask for confirmation if replying to mail.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
