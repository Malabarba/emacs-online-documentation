gnus-discouraged-post-methods is a variable defined in `gnus-msg.el'.
Its value is 
(nndraft nnml nnimap nnmaildir nnmh nnfolder nndir)


Documentation:
A list of back ends that are not used in "real" newsgroups.
This variable is used only when `gnus-post-method' is `current'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
