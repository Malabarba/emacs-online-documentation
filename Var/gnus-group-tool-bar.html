gnus-group-tool-bar is a variable defined in `gnus-group.el'.
Its value is 
gnus-group-tool-bar-retro


Documentation:
Specifies the Gnus group tool bar.

It can be either a list or a symbol referring to a list. See
`gmm-tool-bar-from-list' for the format of the list. The
default key map is `gnus-group-mode-map'.

Pre-defined symbols include `gnus-group-tool-bar-gnome' and
`gnus-group-tool-bar-retro'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
