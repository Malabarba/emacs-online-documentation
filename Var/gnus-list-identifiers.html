--- title: "gnus-list-identifiers" layout: doc --- gnus-list-identifiers is a variable defined in `gnus-sum.el'.
Its value is 
nil


Documentation:
Regexp that matches list identifiers to be removed from subject.
This can also be a list of regexps.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 21.1 of Emacs.
