--- title: "gnus-save-newsrc-file-check-timestamp" layout: doc --- gnus-save-newsrc-file-check-timestamp is a variable defined in `gnus-start.el'.
Its value is 
nil


Documentation:
Check the modification time of the newsrc.eld file before saving it.
When the newsrc.eld file is updated by multiple machines,
checking the file's modification time is a good way to avoid
overwriting updated data.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
