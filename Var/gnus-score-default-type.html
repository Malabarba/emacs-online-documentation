gnus-score-default-type is a variable defined in `gnus-score.el'.
Its value is nil

Documentation:
Default match type when entering new scores.

Should be one of the following symbols.

s: substring
e: exact string
f: fuzzy string
r: regexp string
b: before date
a: after date
n: this date
<: less than number
>: greater than number
=: equal to number

If nil, the user will be asked for a match type.

You can customize this variable.
