--- title: "gnus-summary-next-group-on-exit" layout: doc --- gnus-summary-next-group-on-exit is a variable defined in `gnus-sum.el'.
Its value is 
t


Documentation:
If non-nil, go to the next unread newsgroup on summary exit.
See `gnus-group-goto-unread'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
