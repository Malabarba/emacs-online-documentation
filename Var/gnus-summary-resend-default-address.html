gnus-summary-resend-default-address is a variable defined in `gnus-msg.el'.
Its value is 
t


Documentation:
If non-nil, Gnus tries to suggest a default address to resend to.
If nil, the address field will always be empty after invoking
`gnus-summary-resend-message'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
