--- title: "gnus-use-byte-compile" layout: doc --- gnus-use-byte-compile is a variable defined in `gnus-util.el'.
Its value is 
t


Documentation:
If non-nil, byte-compile crucial run-time code.
Setting it to nil has no effect after the first time `gnus-byte-compile'
is run.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
