gud-inhibit-global-bindings is a variable defined in `edebug.el'.
Its value is nil

This variable is an alias for `edebug-inhibit-emacs-lisp-mode-bindings'.
This variable is obsolete since 24.3;
use `edebug-inhibit-emacs-lisp-mode-bindings' instead.

Documentation:
If non-nil, inhibit Edebug bindings on the C-x C-a key.
By default, loading the `edebug' library causes these bindings to
be installed in `emacs-lisp-mode-map'.
