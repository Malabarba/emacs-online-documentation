help-fns-describe-function-functions is a variable defined in `help-fns.el'.
Its value is 


(eieio-help-constructor eieio-help-generic help-fns--compiler-macro help-fns--parent-mode help-fns--interactive-only help-fns--obsolete)

  • This variable may be risky if used as a file-local variable.

Documentation:
List of functions to run in help buffer in `describe-function'.
Those functions will be run after the header line and argument
list was inserted, and before the documentation will be inserted.
The functions will receive the function name as argument.
