hfy-find-cmd is a variable defined in `htmlfontify.el'.
Its value is 


"find . -type f \\! -name \\*~ \\! -name \\*.flc \\! -path \\*/CVS/\\*"


Documentation:
Find command used to harvest a list of files to attempt to fontify.

You can customize this variable.
