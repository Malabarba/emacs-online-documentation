hfy-init-kludge-hooks is a variable defined in `htmlfontify.el'.
Its value is 
(hfy-kludge-cperl-mode)

  • This variable is an alias for `hfy-init-kludge-hook'.
  • This variable is obsolete since 23.2;
    use `hfy-init-kludge-hook' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
List of functions to call when starting `htmlfontify-buffer' to do any
kludging necessary to get highlighting modes to behave as you want, even
when not running under a window system.

You can customize this variable.
