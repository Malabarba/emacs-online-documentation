hfy-post-html-hooks is a variable defined in `htmlfontify.el'.
Its value is nil

This variable is an alias for `hfy-post-html-hook'.
This variable is obsolete since 24.3;
use `hfy-post-html-hook' instead.
This variable may be risky if used as a file-local variable.

Documentation:
List of functions to call after creating and filling the HTML buffer.
These functions will be called with the HTML buffer as the current buffer.

You can customize this variable.
