--- title: "hi-lock-auto-select-face" layout: doc --- hi-lock-auto-select-face is a variable defined in `hi-lock.el'.
Its value is 
nil


Documentation:
Non-nil means highlighting commands do not prompt for the face to use.
Instead, each hi-lock command will cycle through the faces in
`hi-lock-face-defaults'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
