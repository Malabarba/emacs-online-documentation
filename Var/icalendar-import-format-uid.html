--- title: "icalendar-import-format-uid" layout: doc --- icalendar-import-format-uid is a variable defined in `icalendar.el'.
Its value is 
"
 UID: %s"


Documentation:
Format string defining how the UID element is formatted.
This applies only if the UID is not empty! `%s' is replaced by
the UID.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
