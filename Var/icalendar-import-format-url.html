--- title: "icalendar-import-format-url" layout: doc --- icalendar-import-format-url is a variable defined in `icalendar.el'.
Its value is 
"
 URL: %s"


Documentation:
Format string defining how the URL element is formatted.
This applies only if the URL is not empty! `%s' is replaced by
the URL.

You can customize this variable.
