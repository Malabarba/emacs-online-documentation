ido-cache-ftp-work-directory-time is a variable defined in `ido.el'.
Its value is 
1.0


Documentation:
Maximum time to cache contents of an FTP directory (in hours).

Uses keymap `ido-file-completion-map', which is not currently defined.

Use M-x ido-reread-directory in prompt to refresh list.
If zero, FTP directories are not cached.

You can customize this variable.
