ido-cache-unc-host-shares-time is a variable defined in `ido.el'.
Its value is 
8.0


Documentation:
Maximum time to cache shares of an UNC host (in hours).

Uses keymap `ido-file-completion-map', which is not currently defined.

Use M-x ido-reread-directory in prompt to refresh list.
If zero, UNC host shares are not cached.

You can customize this variable.
