inferior-octave-startup-hook is a variable defined in `octave.el'.
Its value is 
nil

  • This variable is an alias for `inferior-octave-mode-hook'.
  • This variable is obsolete since 24.4;
    use `inferior-octave-mode-hook' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering Inferior Octave mode.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)

You can customize this variable.
