inhibit-first-line-modes-suffixes is a variable defined in `files.el'.
Its value is 


("\\.dz\\'" "\\.xz\\'" "\\.lzma\\'" "\\.lz\\'" "\\.g?z\\'" "\\.bz2\\'" "\\.Z\\'")

  • This variable is an alias for `inhibit-local-variables-suffixes'.
  • This variable is obsolete since 24.1;
    use `inhibit-local-variables-suffixes' instead.

Documentation:
List of regexps matching suffixes to remove from file names.
The function `inhibit-local-variables-p' uses this: when checking
a file name, it first discards from the end of the name anything that
matches one of these regexps.
