initial-calendar-window-hook is a variable defined in `calendar.el'.
Its value is nil

This variable is an alias for `calendar-initial-window-hook'.
This variable is obsolete since 23.1;
use `calendar-initial-window-hook' instead.
This variable may be risky if used as a file-local variable.

Documentation:
List of functions to be called when the calendar window is created.
Quitting the calendar and re-entering it will cause these functions
to be called again.

You can customize this variable.
