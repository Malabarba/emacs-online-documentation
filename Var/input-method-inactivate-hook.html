input-method-inactivate-hook is a variable defined in `mule-cmds.el'.
Its value is nil

This variable is an alias for `input-method-deactivate-hook'.
This variable is obsolete since 24.3;
use `input-method-deactivate-hook' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Normal hook run just after an input method is deactivated.

The variable `current-input-method' still keeps the input method name
just deactivated.

You can customize this variable.
