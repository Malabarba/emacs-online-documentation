isearch-lazy-highlight-cleanup is a variable defined in `isearch.el'.
Its value is t

This variable is an alias for `lazy-highlight-cleanup'.
This variable is obsolete since 22.1;
use `lazy-highlight-cleanup' instead.

Documentation:
Controls whether to remove extra highlighting after a search.
If this is nil, extra highlighting can be "manually" removed with
M-x lazy-highlight-cleanup.

You can customize this variable.
