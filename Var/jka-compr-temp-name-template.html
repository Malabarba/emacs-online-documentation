--- title: "jka-compr-temp-name-template" layout: doc --- jka-compr-temp-name-template is a variable defined in `jka-compr.el'.
Its value is 
"/tmp/jka-com"


Documentation:
Prefix added to all temp files created by jka-compr.
There should be no more than seven characters after the final `/'.

You can customize this variable.
