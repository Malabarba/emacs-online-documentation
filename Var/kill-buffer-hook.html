kill-buffer-hook is a variable defined in `files.el'.
Its value is
(reftex-kill-buffer-hook browse-url-delete-temp-file vc-kill-buffer-hook)


This variable's value is permanent if it is given a local binding.
This variable may be risky if used as a file-local variable.

Documentation:
Hook run when a buffer is killed.
The buffer being killed is current while the hook is running.
See `kill-buffer'.
