lisp-var-doc-command is a variable defined in `inf-lisp.el'.
Its value is 


"(let ((v '%s))\n     (format t \"Documentation for ~a:~&~a\"\n	     v (documentation v 'variable))\n     (values))\n"

  • This variable may be risky if used as a file-local variable.

Documentation:
Command to query inferior Lisp for a variable's documentation.
