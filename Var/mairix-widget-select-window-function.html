--- title: "mairix-widget-select-window-function" layout: doc --- mairix-widget-select-window-function is a variable defined in `mairix.el'.
Its value is 
(lambda nil (select-window (get-largest-window)))

  • This variable may be risky if used as a file-local variable.

Documentation:
Function for selecting the window for customizing the mairix query.
The default chooses the largest window in the current frame.

You can customize this variable.
