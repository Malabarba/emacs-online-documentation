--- title: "makefile-pickup-everything-picks-up-filenames-p" layout: doc --- makefile-pickup-everything-picks-up-filenames-p is a variable defined in `make-mode.el'.
Its value is 
nil


Documentation:
If non-nil, `makefile-pickup-everything' picks up filenames as targets.
This means it calls `makefile-pickup-filenames-as-targets'.
Otherwise filenames are omitted.

You can customize this variable.
