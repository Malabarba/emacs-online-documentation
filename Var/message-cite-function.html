--- title: "message-cite-function" layout: doc --- message-cite-function is a variable defined in `message.el'.
Its value is 
message-cite-original-without-signature

  • This variable may be risky if used as a file-local variable.

Documentation:
*Function for citing an original message.
Predefined functions include `message-cite-original' and
`message-cite-original-without-signature'.
Note that these functions use `mail-citation-hook' if that is non-nil.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.3 of Emacs.
