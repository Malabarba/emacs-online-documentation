--- title: "message-form-letter-separator" layout: doc --- message-form-letter-separator is a variable defined in `message.el'.
Its value is 


"\n----------next form letter message follows this line----------\n"


Documentation:
Separator for `message-send-form-letter'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
