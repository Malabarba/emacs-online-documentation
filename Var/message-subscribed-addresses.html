--- title: "message-subscribed-addresses" layout: doc --- message-subscribed-addresses is a variable defined in `message.el'.
Its value is 
nil


Documentation:
*Specifies a list of addresses the user is subscribed to.
If nil, do not use any predefined list subscriptions. This list of
addresses can be used in conjunction with
`message-subscribed-address-functions' and `message-subscribed-regexps'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
