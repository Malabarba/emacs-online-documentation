message-tool-bar-zap-list is a variable defined in `message.el'.
Its value is 


(new-file open-file dired kill-buffer write-file print-buffer customize help)


Documentation:
List of icon items from the global tool bar.
These items are not displayed on the message mode tool bar.

See `gmm-tool-bar-from-list' for the format of the list.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
