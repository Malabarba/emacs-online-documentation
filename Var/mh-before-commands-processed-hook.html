mh-before-commands-processed-hook is a variable defined in `mh-e.el'.
Its value is nil

This variable may be risky if used as a file-local variable.

Documentation:
Hook run by x before performing outstanding refile and delete requests.

Variables that are useful in this hook include `mh-delete-list'
and `mh-refile-list' which can be used to see which changes will
be made to the current folder, `mh-current-folder'.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 8.0 of the MH-E package that is part of Emacs 22.1.
