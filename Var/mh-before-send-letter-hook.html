--- title: "mh-before-send-letter-hook" layout: doc --- mh-before-send-letter-hook is a variable defined in `mh-e.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run at the beginning of the C-c C-c command.

For example, if you want to check your spelling in your message
before sending, add the `ispell-message' function.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 6.0 of the MH-E package that is part of Emacs 22.1.
