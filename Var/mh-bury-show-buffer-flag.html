--- title: "mh-bury-show-buffer-flag" layout: doc --- mh-bury-show-buffer-flag is a variable defined in `mh-e.el'.
Its value is 
t


Documentation:
*Non-nil means show buffer is buried.

One advantage of not burying the show buffer is that one can
delete the show buffer more easily in an electric buffer list
because of its proximity to its associated MH-Folder buffer. Try
running M-x electric-buffer-list to see what I mean.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 7.0 of the MH-E package that is part of Emacs 22.1.
