mh-compose-skipped-header-fields is a variable defined in `mh-e.el'.
Its value is
("From" "Organization" "References" "In-Reply-To" "X-Face" "Face" "X-Image-URL" "X-Mailer")


Documentation:
List of header fields to skip over when navigating in draft.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 7.4 of the MH-E package that is part of Emacs 22.1.
