mh-delete-msg-hook is a variable defined in `mh-e.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run by M-x mh-delete-msg after marking each message for deletion.

For example, a past maintainer of MH-E used this once when he
kept statistics on his mail usage.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 6.0 of the MH-E package that is part of Emacs 22.1.
