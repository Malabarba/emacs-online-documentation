mh-insert-signature-hook is a variable defined in `mh-e.el'.
Its value is nil

This variable may be risky if used as a file-local variable.

Documentation:
Hook run by C-c C-s after signature has been inserted.

Hook functions may access the actual name of the file or the
function used to insert the signature with
`mh-signature-file-name'.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 8.0 of the MH-E package that is part of Emacs 22.1.
