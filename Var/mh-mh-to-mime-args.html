--- title: "mh-mh-to-mime-args" layout: doc --- mh-mh-to-mime-args is a variable defined in `mh-mime.el'.
Its value is 
nil


Documentation:
Extra arguments for M-x mh-mh-to-mime to pass to the "mhbuild" command.
The arguments are passed to "mhbuild" if M-x mh-mh-to-mime is
given a prefix argument. Normally default arguments to
"mhbuild" are specified in the MH profile.
