mh-pack-folder-hook is a variable defined in `mh-e.el'.
Its value is nil

This variable may be risky if used as a file-local variable.

Documentation:
Hook run by F p after renumbering the messages.
Hook functions can access the current folder name with `mh-current-folder'.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 8.2 of the MH-E package.
