mh-quit-hook is a variable defined in `mh-e.el'.
Its value is nil

This variable may be risky if used as a file-local variable.

Documentation:
Hook run by q after quitting MH-E.

This hook is not run in an MH-E context, so you might use it to
modify the window setup.

See also `mh-before-quit-hook'.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 6.0 of the MH-E package that is part of Emacs 22.1.
