--- title: "mh-recenter-summary-flag" layout: doc --- mh-recenter-summary-flag is a variable defined in `mh-e.el'.
Its value is 
nil


Documentation:
*Non-nil means to recenter the summary window.

If this option is turned on, recenter the summary window when the
show window is toggled off.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 7.0 of the MH-E package that is part of Emacs 22.1.
