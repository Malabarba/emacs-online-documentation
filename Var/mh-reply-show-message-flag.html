--- title: "mh-reply-show-message-flag" layout: doc --- mh-reply-show-message-flag is a variable defined in `mh-e.el'.
Its value is 
t


Documentation:
*Non-nil means the MH-Show buffer is displayed when replying.

If you include the message automatically, you can hide the
MH-Show buffer by turning off this option.

See also `mh-reply'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 7.0 of the MH-E package that is part of Emacs 22.1.
