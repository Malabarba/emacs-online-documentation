mh-search-mode-hook is a variable defined in `mh-e.el'.
Its value is nil

This variable may be risky if used as a file-local variable.

Documentation:
Hook run upon entry to `mh-search-mode'.

If you find that you do the same thing over and over when editing
the search template, you may wish to bind some shortcuts to keys.
This can be done with this hook which is called when
F s is run on a new pattern.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 8.0 of the MH-E package that is part of Emacs 22.1.
