--- title: "mh-show-hook" layout: doc --- mh-show-hook is a variable defined in `mh-e.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after RET shows a message.

It is the last thing called after messages are displayed. It's
used to affect the behavior of MH-E in general or when
`mh-show-mode-hook' is too early. See `mh-show-mode-hook'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 6.0 of the MH-E package that is part of Emacs 22.1.
