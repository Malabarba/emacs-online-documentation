--- title: "mh-ticked-messages-folders" layout: doc --- mh-ticked-messages-folders is a variable defined in `mh-e.el'.
Its value is 
t


Documentation:
Folders searched for `mh-tick-seq'.

Set this option to "Inbox" to search the "+inbox" folder or
"All" to search all of the top level folders. Otherwise, list
the folders that should be searched with the "Choose Folders"
menu item.

See also `mh-recursive-folders-flag'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 8.0 of the MH-E package that is part of Emacs 22.1.
