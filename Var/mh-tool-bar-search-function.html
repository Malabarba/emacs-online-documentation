mh-tool-bar-search-function is a variable defined in `mh-e.el'.
Its value is 
mh-search

  • This variable may be risky if used as a file-local variable.

Documentation:
*Function called by the tool bar search button.

By default, this is set to `mh-search'. You can also choose
"Other Function" from the "Value Menu" and enter a function
of your own choosing.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 7.0 of the MH-E package that is part of Emacs 22.1.
