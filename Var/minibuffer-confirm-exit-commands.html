minibuffer-confirm-exit-commands is a variable defined in `minibuffer.el'.
Its value is 


(completion-at-point minibuffer-complete minibuffer-complete-word PC-complete PC-complete-word)

  • This variable may be risky if used as a file-local variable.

Documentation:
A list of commands which cause an immediately following
`minibuffer-complete-and-exit' to ask for extra confirmation.
