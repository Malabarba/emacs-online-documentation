--- title: "mm-inline-text-html-with-images" layout: doc --- mm-inline-text-html-with-images is a variable defined in `mm-decode.el'.
Its value is 
nil


Documentation:
If non-nil, Gnus will allow retrieving images in HTML that has tags.
See also the documentation for the `mm-w3m-safe-url-regexp'
variable.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
