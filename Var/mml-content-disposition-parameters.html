--- title: "mml-content-disposition-parameters" layout: doc --- mml-content-disposition-parameters is a variable defined in `mml.el'.
Its value is 
(filename creation-date modification-date read-date)


Documentation:
*A list of acceptable parameters in MML tag.
These parameters are generated in Content-Disposition header if exists.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
