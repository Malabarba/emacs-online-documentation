mpc-songs-format is a variable defined in `mpc.el'.
Its value is 


"%2{Disc--}%3{Track} %-5{Time} %25{Title} %20{Album} %20{Artist} %5{Date}"


Documentation:
Format used to display each song in the list of songs.

You can customize this variable.
