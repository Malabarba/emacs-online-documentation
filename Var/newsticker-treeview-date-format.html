--- title: "newsticker-treeview-date-format" layout: doc --- newsticker-treeview-date-format is a variable defined in `newst-treeview.el'.
Its value is 
"%d.%m.%y, %H:%M"


Documentation:
Format for the date column in the treeview list buffer.
See `format-time-string' for a list of valid specifiers.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
