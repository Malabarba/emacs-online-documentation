--- title: "next-error-recenter" layout: doc --- next-error-recenter is a variable defined in `simple.el'.
Its value is 
nil


Documentation:
Display the line in the visited source file recentered as specified.
If non-nil, the value is passed directly to `recenter'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
