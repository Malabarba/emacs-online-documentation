--- title: "nnimap-streaming" layout: doc --- nnimap-streaming is a variable defined in `nnimap.el'.
Its value is 
t


Documentation:
If non-nil, try to use streaming commands with IMAP servers.
Switching this off will make nnimap slower, but it helps with
some servers.

This is a Gnus server variable. See Info node `(gnus)Select Methods'.
