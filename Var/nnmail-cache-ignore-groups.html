--- title: "nnmail-cache-ignore-groups" layout: doc --- nnmail-cache-ignore-groups is a variable defined in `nnmail.el'.
Its value is 
nil


Documentation:
Regexp that matches group names to be ignored when inserting message ids into the cache (`nnmail-cache-insert').
This can also be a list of regexps.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
