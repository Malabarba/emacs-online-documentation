--- title: "nnmail-split-header-length-limit" layout: doc --- nnmail-split-header-length-limit is a variable defined in `nnmail.el'.
Its value is 
2048


Documentation:
Header lines longer than this limit are excluded from the split function.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 21.1 of Emacs.
