--- title: "nroff-electric-mode-hook" layout: doc --- nroff-electric-mode-hook is a variable defined in `nroff-mode.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering or leaving `nroff-electric-mode'.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)
