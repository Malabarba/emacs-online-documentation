nslookup-font-lock-keywords is a variable defined in `net-utils.el'.
Its value is 


(("^[A-Za-z0-9 _]+:" 0 font-lock-type-face)
 ("\\<\\(SOA\\|NS\\|MX\\|A\\|CNAME\\)\\>" 1 font-lock-keyword-face)
 ("[0-9]+\\.[0-9]+\\.[0-9]+\\.[0-9]+" 0 font-lock-variable-name-face)
 ("[-A-Za-z0-9]+\\.[-A-Za-z0-9]+\\(\\.[-A-Za-z0-9]+\\)*" 0 font-lock-variable-name-face))

  • This variable may be risky if used as a file-local variable.
