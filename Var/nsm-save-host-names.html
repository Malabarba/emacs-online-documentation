--- title: "nsm-save-host-names" layout: doc --- nsm-save-host-names is a variable defined in `nsm.el'.
Its value is 
nil


Documentation:
If non-nil, always save host names in the structures in `nsm-settings-file'.
By default, only hosts that have exceptions have their names
stored in plain text.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
