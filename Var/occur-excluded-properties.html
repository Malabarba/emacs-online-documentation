occur-excluded-properties is a variable defined in `replace.el'.
Its value is
(read-only invisible intangible field mouse-face help-echo local-map keymap yank-handler follow-link)


Documentation:
Text properties to discard when copying lines to the *Occur* buffer.
The value should be a list of text properties to discard or t,
which means to discard all text properties.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 22.1 of Emacs.
