--- title: "org-agenda-follow-indirect" layout: doc --- org-agenda-follow-indirect is a variable defined in `org-agenda.el'.
Its value is 
nil


Documentation:
Non-nil means `org-agenda-follow-mode' displays only the
current item's tree, in an indirect buffer.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.1 of Emacs.
