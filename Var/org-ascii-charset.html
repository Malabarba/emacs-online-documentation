--- title: "org-ascii-charset" layout: doc --- org-ascii-charset is a variable defined in `ox-ascii.el'.
Its value is 
ascii


Documentation:
The charset allowed to represent various elements and objects.
Possible values are:
`ascii' Only use plain ASCII characters
`latin1' Include Latin-1 characters
`utf-8' Use all UTF-8 characters

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
