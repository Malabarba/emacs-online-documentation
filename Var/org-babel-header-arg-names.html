org-babel-header-arg-names is a variable defined in `ob.el'.
Its value is
(cache cmdline colnames comments dir eval exports file file-desc hlines mkdirp no-expand noeval noweb noweb-ref noweb-sep padline results rownames sep session shebang tangle var wrap)


This variable may be risky if used as a file-local variable.

Documentation:
Common header arguments used by org-babel.
Note that individual languages may define their own language
specific header arguments as well.
