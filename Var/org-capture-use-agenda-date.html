--- title: "org-capture-use-agenda-date" layout: doc --- org-capture-use-agenda-date is a variable defined in `org-capture.el'.
Its value is 
nil


Documentation:
Non-nil means use the date at point when capturing from agendas.
When nil, you can still capture using the date at point with M-x org-agenda-capture.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
