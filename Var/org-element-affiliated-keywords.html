org-element-affiliated-keywords is a variable defined in `org-element.el'.
Its value is 


("CAPTION" "DATA" "HEADER" "HEADERS" "LABEL" "NAME" "PLOT" "RESNAME" "RESULT" "RESULTS" "SOURCE" "SRCNAME" "TBLNAME")

  • This variable may be risky if used as a file-local variable.

Documentation:
List of affiliated keywords as strings.
By default, all keywords setting attributes (i.e. "ATTR_LATEX")
are affiliated keywords and need not to be in this list.
