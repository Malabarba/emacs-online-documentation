--- title: "org-export-in-background" layout: doc --- org-export-in-background is a variable defined in `ox.el'.
Its value is 
nil


Documentation:
Non-nil means export and publishing commands will run in background.
Results from an asynchronous export are never displayed
automatically. But you can retrieve them with M-x org-export-stack.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
