--- title: "org-html-allow-name-attribute-in-anchors" layout: doc --- org-html-allow-name-attribute-in-anchors is a variable defined in `ox-html.el'.
Its value is 
t


Documentation:
When nil, do not set "name" attribute in anchors.
By default, anchors are formatted with both "id" and "name"
attributes, when appropriate.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
