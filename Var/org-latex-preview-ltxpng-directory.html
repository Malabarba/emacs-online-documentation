--- title: "org-latex-preview-ltxpng-directory" layout: doc --- org-latex-preview-ltxpng-directory is a variable defined in `org.el'.
Its value is 
"ltxpng/"


Documentation:
Path to store latex preview images.
A relative path here creates many directories relative to the
processed org files paths. An absolute path puts all preview
images at the same place.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
