--- title: "org-link-search-must-match-exact-headline" layout: doc --- org-link-search-must-match-exact-headline is a variable defined in `org.el'.
Its value is 
query-to-create


Documentation:
Non-nil means internal links in Org files must exactly match a headline.
When nil, the link search tries to match a phrase with all words
in the search text.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.1 of Emacs.
