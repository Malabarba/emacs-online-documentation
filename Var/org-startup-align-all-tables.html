org-startup-align-all-tables is a variable defined in `org.el'.
Its value is nil

Documentation:
Non-nil means align all tables when visiting a file.
This is useful when the column width in tables is forced with cookies
in table fields. Such tables will look correct only after the first re-align.
This can also be configured on a per-file basis by adding one of
the following lines anywhere in the buffer:
#+STARTUP: align
#+STARTUP: noalign

You can customize this variable.
