--- title: "org-time-clocksum-use-fractional" layout: doc --- org-time-clocksum-use-fractional is a variable defined in `org.el'.
Its value is 
nil


Documentation:
When non-nil, M-x org-clock-display uses fractional times.
See `org-time-clocksum-format' for more on time clock formats.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
