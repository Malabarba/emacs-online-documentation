--- title: "org-use-last-clock-out-time-as-effective-time" layout: doc --- org-use-last-clock-out-time-as-effective-time is a variable defined in `org.el'.
Its value is 
nil


Documentation:
When non-nil, use the last clock out time for `org-todo'.
Note that this option has precedence over the combined use of
`org-use-effective-time' and `org-extend-today-until'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
