pascal-toggle-completions is a variable defined in `pascal.el'.
Its value is nil

This variable is obsolete since 24.1;
use `completion-cycle-threshold' instead.

Documentation:
If non-nil, `pascal-complete-word' tries all possible completions.
Repeated use of M-x pascal-complete-word then shows all
completions in turn, instead of displaying a list of all possible
completions.
