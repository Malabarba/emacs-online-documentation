--- title: "profiler-report-mode-hook" layout: doc --- profiler-report-mode-hook is a variable defined in `profiler.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering Profiler-Report mode.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)
