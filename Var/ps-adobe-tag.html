--- title: "ps-adobe-tag" layout: doc --- ps-adobe-tag is a variable defined in `ps-print.el'.
Its value is 
"%!PS-Adobe-3.0
"


Documentation:
Contains the header line identifying the output as PostScript.
By default, `ps-adobe-tag' contains the standard identifier. Some printers
require slightly different versions of this line.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
