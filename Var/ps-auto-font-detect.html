ps-auto-font-detect is a variable defined in `ps-print.el'.
Its value is 
t


Documentation:
Non-nil means automatically detect bold/italic/underline face attributes.
If nil, we rely solely on the lists `ps-bold-faces', `ps-italic-faces', and
`ps-underlined-faces'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
