--- title: "ps-postscript-code-directory" layout: doc --- ps-postscript-code-directory is a variable defined in `ps-print.el'.
Its value is 
"/usr/local/share/emacs/25.0.50/etc/"


Documentation:
Directory where it's located the PostScript prologue file used by ps-print.
By default, this directory is the same as in the variable `data-directory'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
