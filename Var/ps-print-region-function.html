--- title: "ps-print-region-function" layout: doc --- ps-print-region-function is a variable defined in `ps-print.el'.
Its value is 
call-process-region

  • This variable may be risky if used as a file-local variable.

Documentation:
Specify a function to print the region on a PostScript printer.
See definition of `call-process-region' for calling conventions. The fourth
and the sixth arguments are both nil.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
