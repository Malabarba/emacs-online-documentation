--- title: "ps-show-n-of-n" layout: doc --- ps-show-n-of-n is a variable defined in `ps-print.el'.
Its value is 
t


Documentation:
Non-nil means show page numbers as N/M, meaning page N of M.
NOTE: page numbers are displayed as part of headers,
see variable `ps-print-header'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
