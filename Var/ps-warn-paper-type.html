--- title: "ps-warn-paper-type" layout: doc --- ps-warn-paper-type is a variable defined in `ps-print.el'.
Its value is 
t


Documentation:
Non-nil means give an error if paper size is not equal to `ps-paper-type'.

It's used when `ps-spool-config' is set to `setpagedevice'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20 of Emacs.
