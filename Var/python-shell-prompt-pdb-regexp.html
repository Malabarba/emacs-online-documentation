--- title: "python-shell-prompt-pdb-regexp" layout: doc --- python-shell-prompt-pdb-regexp is a variable defined in `python.el'.
Its value is 
"[(<]*[Ii]?[Pp]db[>)]+ "


Documentation:
Regular expression matching pdb input prompt of Python shell.
It should not contain a caret (^) at the beginning.

You can customize this variable.
