python-use-skeletons is a variable defined in `python.el'.
Its value is nil

This variable is an alias for `python-skeleton-autoinsert'.
This variable is obsolete since 24.3;
use `python-skeleton-autoinsert' instead.

Documentation:
Non-nil means template skeletons will be automagically inserted.
This happens when pressing "if", for example, to prompt for
the if condition.

You can customize this variable.
