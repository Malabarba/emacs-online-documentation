rcirc-print-hooks is a variable defined in `rcirc.el'.
Its value is nil

This variable is an alias for `rcirc-print-functions'.
This variable is obsolete since 24.3;
use `rcirc-print-functions' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Hook run after text is printed.
Called with 5 arguments, PROCESS, SENDER, RESPONSE, TARGET and TEXT.

You can customize this variable.
