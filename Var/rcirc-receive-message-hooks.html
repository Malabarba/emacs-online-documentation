rcirc-receive-message-hooks is a variable defined in `rcirc.el'.
Its value is nil

This variable is an alias for `rcirc-receive-message-functions'.
This variable is obsolete since 24.3;
use `rcirc-receive-message-functions' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Hook functions run when a message is received from server.
Function is called with PROCESS, COMMAND, SENDER, ARGS and LINE.
