--- title: "rcirc-reconnect-delay" layout: doc --- rcirc-reconnect-delay is a variable defined in `rcirc.el'.
Its value is 
0


Documentation:
The minimum interval in seconds between reconnect attempts.
When 0, do not auto-reconnect.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
