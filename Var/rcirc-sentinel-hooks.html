rcirc-sentinel-hooks is a variable defined in `rcirc.el'.
Its value is nil

This variable is an alias for `rcirc-sentinel-functions'.
This variable is obsolete since 24.3;
use `rcirc-sentinel-functions' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Hook functions called when the process sentinel is called.
Functions are called with PROCESS and SENTINEL arguments.
