recenter-redisplay is a variable defined in `C source code'.
Its value is 
tty


Documentation:
Non-nil means `recenter' redraws entire frame.
If this option is non-nil, then the `recenter' command with a nil
argument will redraw the entire frame; the special value `tty' causes
the frame to be redrawn only if it is a tty frame.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
