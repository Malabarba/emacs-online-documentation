reftex-select-bib-map is a variable defined in `reftex-sel.el'.
Its value is shown below.
  • This variable is an alias for `reftex-select-bib-mode-map'.
  • This variable is obsolete since 24.1;
    use `reftex-select-bib-mode-map' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
Keymap used for *RefTeX Select* buffer, when selecting a BibTeX entry.
This keymap can be used to configure the BibTeX selection process which is
started with the command M-x reftex-citation.
