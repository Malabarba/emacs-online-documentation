--- title: "reftex-toc-include-index-entries" layout: doc --- reftex-toc-include-index-entries is a variable defined in `reftex-vars.el'.
Its value is 
nil


Documentation:
Non-nil means, include index entries in *toc* buffer.
This flag can be toggled from within the *toc* buffer with the `i' key.

You can customize this variable.
