--- title: "remember-directory-file-name-format" layout: doc --- remember-directory-file-name-format is a variable defined in `remember.el'.
Its value is 
"%Y-%m-%d_%T-%z"


Documentation:
Format string for the file name in which to store unprocessed data.
This is passed to `format-time-string'.
Used by `remember-store-in-files'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
