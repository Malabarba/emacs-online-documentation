--- title: "remember-notes-initial-major-mode" layout: doc --- remember-notes-initial-major-mode is a variable defined in `remember.el'.
Its value is 
nil


Documentation:
Major mode to use in the notes buffer when it's created.
If this is nil, use `initial-major-mode'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
