--- title: "rmail-current-message" layout: doc --- rmail-current-message is a variable defined in `rmail.el'.
Its value is 
nil

  • This variable's value is permanent if it is given a local binding.

Documentation:
Integer specifying the message currently being displayed in this folder.
Counts messages from 1 to `rmail-total-messages'. A value of 0
means there are no messages in the folder.
