rmail-movemail-flags is a variable defined in `rmail.el'.
Its value is 
nil


Documentation:
List of flags to pass to movemail.
Most commonly used to specify `-g' to enable GSS-API authentication
or `-k' to enable Kerberos authentication.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 20.3 of Emacs.
