rmail-remote-password-error is a variable defined in `rmail.el'.
Its value is 


"invalid usercode or password\\|\nunknown user name or bad password\\|Authentication failed\\|MU_ERR_AUTH_FAILURE"


Documentation:
Regular expression matching incorrect-password POP or IMAP server error
messages.
If you get an incorrect-password error that this expression does not match,
please report it with M-x report-emacs-bug.
