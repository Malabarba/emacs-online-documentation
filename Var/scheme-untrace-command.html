--- title: "scheme-untrace-command" layout: doc --- scheme-untrace-command is a variable defined in `cmuscheme.el'.
Its value is 
"(untrace %s)"

  • This variable may be risky if used as a file-local variable.

Documentation:
Template for switching off tracing of a Scheme procedure.
Scheme 48 and Scsh users should set this variable to ",untrace %s".

You can customize this variable.
