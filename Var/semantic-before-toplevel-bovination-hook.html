--- title: "semantic-before-toplevel-bovination-hook" layout: doc --- semantic-before-toplevel-bovination-hook is a variable defined in `semantic.el'.
Its value is 
nil

  • This variable is an alias for `semantic--before-fetch-tags-hook'.
  • This variable is obsolete since 23.2;
    use `semantic--before-fetch-tags-hook' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
Hooks run before a buffer is parsed for tags.
It is called before any request for tags is made via the function
`semantic-fetch-tags' by an application.
If any hook returns a nil value, the cached value is returned
immediately, even if it is empty.
