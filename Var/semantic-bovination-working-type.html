semantic-bovination-working-type is a variable defined in `semantic.el'.
Its value is percent

This variable is an alias for `semantic-working-type'.
This variable is obsolete since 23.2;
use `semantic-working-type' instead.

Documentation:
The type of working message to use when parsing.
'percent means we are doing a linear parse through the buffer.
'dynamic means we are reparsing specific tags.
