semantic-edits-delete-change-hooks is a variable defined in `edit.el'.
Its value is nil

This variable is an alias for `semantic-edits-delete-change-functions'.
This variable is obsolete since 24.3;
use `semantic-edits-delete-change-functions' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Abnormal hook run before a change overlay is deleted.
Deleted changes occur when multiple changes are merged.
Functions must take one argument representing an overlay being deleted.
