semantic-edits-reparse-change-hooks is a variable defined in `edit.el'.
Its value is nil

This variable is an alias for `semantic-edits-reparse-change-functions'.
This variable is obsolete since 24.3;
use `semantic-edits-reparse-change-functions' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Abnormal hook run after a change results in a reparse.
Functions are called before the overlay is deleted, and after the
incremental reparse.
