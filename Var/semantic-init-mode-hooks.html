semantic-init-mode-hooks is a variable defined in `semantic.el'.
Its value is nil

Automatically becomes buffer-local when set.
This variable is an alias for `semantic-init-mode-hook'.
This variable is obsolete since 23.2;
use `semantic-init-mode-hook' instead.
This variable may be risky if used as a file-local variable.

Documentation:
Hook run when a buffer of a particular mode is initialized.
