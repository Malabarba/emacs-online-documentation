semantic-lex-reset-hooks is a variable defined in `lex.el'.
Its value is 
nil

  • This variable is an alias for `semantic-lex-reset-functions'.
  • This variable is obsolete since 24.3;
    use `semantic-lex-reset-functions' instead.
  • This variable may be risky if used as a file-local variable.

Documentation:
Abnormal hook used by major-modes to reset lexical analyzers.
Hook functions are called with START and END values for the
current lexical pass. Should be set with `add-hook', specifying
a LOCAL option.
