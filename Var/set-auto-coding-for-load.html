set-auto-coding-for-load is a variable defined in `mule.el'.
Its value is 
nil


Documentation:
Non-nil means respect a "unibyte: t" entry in file local variables.
Emacs binds this variable to t when loading or byte-compiling Emacs Lisp
files.
