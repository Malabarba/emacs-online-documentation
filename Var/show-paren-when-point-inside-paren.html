--- title: "show-paren-when-point-inside-paren" layout: doc --- show-paren-when-point-inside-paren is a variable defined in `paren.el'.
Its value is 
nil


Documentation:
If non-nil, show parens when point is just inside one.
This will only be done when point isn't also just outside a paren.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
