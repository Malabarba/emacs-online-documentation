--- title: "simula-abbrev-stdproc" layout: doc --- simula-abbrev-stdproc is a variable defined in `simula.el'.
Its value is 
abbrev-table


Documentation:
Specify how to convert case for standard SIMULA procedure and class names.
Value is one of the symbols `upcase', `downcase', `capitalize',
(as in) `abbrev-table', or nil if they should not be changed.

You can customize this variable.
