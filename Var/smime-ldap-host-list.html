--- title: "smime-ldap-host-list" layout: doc --- smime-ldap-host-list is a variable defined in `smime.el'.
Its value is 
nil


Documentation:
A list of LDAP hosts with S/MIME user certificates.
If needed search base, binddn, passwd, etc. for the LDAP host
must be set in `ldap-host-parameters-alist'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
