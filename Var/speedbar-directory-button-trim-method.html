--- title: "speedbar-directory-button-trim-method" layout: doc --- speedbar-directory-button-trim-method is a variable defined in `speedbar.el'.
Its value is 
span


Documentation:
Indicates how the directory button will be displayed.
Possible values are:
'span - span large directories over multiple lines.
'trim - trim large directories to only show the last few.
nil - no trimming.

You can customize this variable.
