--- title: "speedbar-vc-directory-enable-hook" layout: doc --- speedbar-vc-directory-enable-hook is a variable defined in `speedbar.el'.
Its value is 
nil

  • This variable may be risky if used as a file-local variable.

Documentation:
Return non-nil if the current directory should be checked for Version Control.
Functions in this hook must accept one parameter which is the directory
being checked.

You can customize this variable.
