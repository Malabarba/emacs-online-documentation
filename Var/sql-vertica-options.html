--- title: "sql-vertica-options" layout: doc --- sql-vertica-options is a variable defined in `sql.el'.
Its value is 
("-P" "pager=off")


Documentation:
List of additional options for `sql-vertica-program'.
The default value disables the internal pager.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 25.1 of Emacs.
