--- title: "starttls-gnutls-program" layout: doc --- starttls-gnutls-program is a variable defined in `starttls.el'.
Its value is 
"gnutls-cli"

  • This variable may be risky if used as a file-local variable.

Documentation:
Name of GnuTLS command line tool.
This program is used when GnuTLS is used, i.e. when
`starttls-use-gnutls' is non-nil.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
