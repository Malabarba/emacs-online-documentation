--- title: "subword-mode-hook" layout: doc --- subword-mode-hook is a variable defined in `subword.el'.
Its value is 
(subword-mode-set-explicitly)

  • This variable may be risky if used as a file-local variable.

Documentation:
Hook run after entering or leaving `subword-mode'.
No problems result if this variable is not bound.
`add-hook' automatically binds it. (This is true for all hook variables.)
