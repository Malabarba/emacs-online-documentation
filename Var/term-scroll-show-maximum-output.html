--- title: "term-scroll-show-maximum-output" layout: doc --- term-scroll-show-maximum-output is a variable defined in `term.el'.
Its value is 
nil

  • This variable's value is permanent if it is given a local binding.

Documentation:
Controls how interpreter output causes window to scroll.
If non-nil, then show the maximum output when the window is scrolled.

See variable `term-scroll-to-bottom-on-output'.
This variable is buffer-local.

You can customize this variable.
