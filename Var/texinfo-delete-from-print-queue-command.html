texinfo-delete-from-print-queue-command is a variable defined in `texinfo.el'.
Its value is "lprm"

This variable may be risky if used as a file-local variable.

Documentation:
Command string used to delete a job from the line printer queue.
Command is used by M-x texinfo-delete-from-print-queue based on
number provided by a previous M-x tex-show-print-queue
command.

You can customize this variable.
