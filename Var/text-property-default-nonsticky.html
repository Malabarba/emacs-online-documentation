text-property-default-nonsticky is a variable defined in `C source code'.
Its value is ((fill-space . t)
(composition . t)
(syntax-table . t)
(display . t))


Documentation:
Alist of properties vs the corresponding non-stickiness.
Each element has the form (PROPERTY . NONSTICKINESS).

If a character in a buffer has PROPERTY, new text inserted adjacent to
the character doesn't inherit PROPERTY if NONSTICKINESS is non-nil,
inherits it if NONSTICKINESS is nil. The `front-sticky' and
`rear-nonsticky' properties of the character override NONSTICKINESS.
