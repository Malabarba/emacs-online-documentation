tls-success is a variable defined in `tls.el'.
Its value is 
"- Handshake was completed\\|SSL handshake has read "


Documentation:
Regular expression indicating completed TLS handshakes.
The default is what GnuTLS's "gnutls-cli" or OpenSSL's
"openssl s_client" outputs.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
