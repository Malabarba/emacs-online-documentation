today-invisible-calendar-hook is a variable defined in `calendar.el'.
Its value is nil

This variable is an alias for `calendar-today-invisible-hook'.
This variable is obsolete since 23.1;
use `calendar-today-invisible-hook' instead.
This variable may be risky if used as a file-local variable.

Documentation:
List of functions called whenever the current date is not visible.
See also `calendar-today-visible-hook'.

You can customize this variable.
