--- title: "tramp-ssh-controlmaster-options" layout: doc --- tramp-ssh-controlmaster-options is a variable defined in `tramp.el'.
Its value is 


"-o ControlPath=%t.%%r@%%h:%%p -o ControlMaster=auto -o ControlPersist=no"

  • This variable may be risky if used as a file-local variable.

Documentation:
Call ssh to detect whether it supports the Control* arguments.
