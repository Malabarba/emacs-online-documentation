tty-menu-open-use-tmm is a variable defined in `menu-bar.el'.
Its value is 
nil


Documentation:
If non-nil, on a TTY will invoke `tmm-menubar'.

If nil, will drop down the menu corresponding to the
first (leftmost) menu-bar item; you can select other items by typing
C-f, C-b, and .

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
