--- title: "uce-default-headers" layout: doc --- uce-default-headers is a variable defined in `uce.el'.
Its value is 
"Errors-To: nobody@localhost
Precedence: bulk
"


Documentation:
Additional headers to use when responding to a UCE with M-x uce-reply-to-uce.
These are mostly meant for headers that prevent delivery errors reporting.

You can customize this variable.
