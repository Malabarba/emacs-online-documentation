vc-cvs-sticky-date-format-string is a variable defined in `vc-cvs.el'.
Its value is 
"%c"


Documentation:
Format string for mode-line display of sticky date.
Format is according to `format-time-string'. Only used if
`vc-cvs-sticky-tag-display' is t.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 22.1 of Emacs.
