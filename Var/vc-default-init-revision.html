vc-default-init-revision is a variable defined in `vc.el'.
Its value is "1.1"

Documentation:
A string used as the default revision number when a new file is registered.
This can be overridden by giving a prefix argument to C-x v i. This
can also be overridden by a particular VC backend.

You can customize this variable.

This variable was introduced, or its default value was changed, in
version 20.3 of Emacs.
