vc-git-diff-switches is a variable defined in `vc-git.el'.
Its value is 
t


Documentation:
String or list of strings specifying switches for Git diff under VC.
If nil, use the value of `vc-diff-switches'. If t, use no switches.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 23.1 of Emacs.
