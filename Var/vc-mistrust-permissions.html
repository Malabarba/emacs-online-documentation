vc-mistrust-permissions is a variable defined in `vc-hooks.el'.
Its value is 
t


Documentation:
If non-nil, don't assume permissions/ownership track version-control status.
If nil, do rely on the permissions.
See also variable `vc-consult-headers'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
