vera-q-values-regexp is a variable defined in `vera-mode.el'.
Its value is 


"\\<\\(\\(?:g\\(?:nr\\|r[01xz]\\)\\|nr\\|r[01xz]\\|s\\(?:nr\\|r[01xz]\\)\\)\\)\\>"

  • This variable may be risky if used as a file-local variable.
