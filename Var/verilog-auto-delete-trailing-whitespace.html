--- title: "verilog-auto-delete-trailing-whitespace" layout: doc --- verilog-auto-delete-trailing-whitespace is a variable defined in `verilog-mode.el'.
Its value is 
nil

  • This variable is safe as a file local variable if its value
    satisfies the predicate `verilog-booleanp'.

Documentation:
Non-nil means to `delete-trailing-whitespace' in `verilog-auto'.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.1 of Emacs.
