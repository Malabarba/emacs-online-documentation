verilog-auto-template-warn-unused is a variable defined in `verilog-mode.el'.
Its value is 
nil

  • This variable is safe as a file local variable if its value
    satisfies the predicate `verilog-booleanp'.

Documentation:
Non-nil means report warning if an AUTO_TEMPLATE line is not used.
This feature is not supported before Emacs 21.1 or XEmacs 21.4.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.3 of Emacs.
