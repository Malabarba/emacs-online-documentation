verilog-case-fold is a variable defined in `verilog-mode.el'.
Its value is 
t

  • This variable is safe as a file local variable if its value
    satisfies the predicate `verilog-booleanp'.

Documentation:
Non-nil means `verilog-mode' regexps should ignore case.
This variable is t for backward compatibility; nil is suggested.

You can customize this variable.

This variable was introduced, or its default value was changed, in version 24.4 of Emacs.
