verilog-toggle-completions is a variable defined in `verilog-mode.el'.
Its value is 
nil


Documentation:
True means C-M-i should try all possible completions one by one.
Repeated use of C-M-i will show you all of them.
Normally, when there is more than one possible completion,
it displays a list of all possible completions.
