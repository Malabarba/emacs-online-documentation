vhdl-stutter-mode is a variable defined in `vhdl-mode.el'.
Its value is nil

Documentation:
Non-nil if Vhdl-Stutter mode is enabled.
See the command `vhdl-stutter-mode' for a description of this minor mode.
Setting this variable directly does not take effect;
either customize it (see the info node `Easy Customization')
or call the function `vhdl-stutter-mode'.
