viper-insert-local-user-minor-mode is a variable defined in `viper-init.el'.
Its value is nil
Local in buffer *scratch*; global value is the same.

Automatically becomes buffer-local when set.

Documentation:
Auxiliary minor mode for buffer-local user-defined bindings in Insert state.
This is a way to overshadow normal Insert mode bindings locally to certain
designated buffers.
(buffer local)
