--- title: "viper-related-files-and-buffers-ring" layout: doc --- viper-related-files-and-buffers-ring is a variable defined in `viper-init.el'.
Its value is 
nil

  • Automatically becomes permanently buffer-local when set.

Documentation:
List of file and buffer names that are considered to be related to the current buffer.
Related buffers can be cycled through via :R and :P commands.

You can customize this variable.
