--- title: "void-text-area-pointer" layout: doc --- void-text-area-pointer is a variable defined in `C source code'.
Its value is 
arrow


Documentation:
The pointer shape to show in void text areas.
A value of nil means to show the text pointer. Other options are
`arrow', `text', `hand', `vdrag', `hdrag', `nhdrag', `modeline', and
`hourglass'.

You can customize this variable.
