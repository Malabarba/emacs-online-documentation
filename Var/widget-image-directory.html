widget-image-directory is a variable defined in `wid-edit.el'.
Its value is 
"/usr/local/share/emacs/25.0.50/etc/images/custom/"


Documentation:
Where widget button images are located.
If this variable is nil, widget will try to locate the directory
automatically.

You can customize this variable.
