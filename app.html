Description

This app contains a convenient list of ALL built-in functions and variables contained in Emacs. Just tap one of them and you'll be sent to its documentation right here.

There are 2 ways of visiting a function's (or variable's) documentation:

  • You can search for a function straight from the app.
  • You can select the function's name when reading it somewhere else (like the browser) and share it to Emacs Documentation.
